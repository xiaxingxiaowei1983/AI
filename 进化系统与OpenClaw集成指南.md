# 进化系统与OpenClaw定时任务集成指南

## 概述

本文档将指导您如何将进化系统与 OpenClaw 定时任务集成，以便您可以通过 OpenClaw 的定时任务界面查看和管理进化报告。

## 前提条件

1. **OpenClaw 已安装并运行**：确保 OpenClaw 网关正在端口 18789 上运行
2. **进化系统已配置**：确保 `evolver` 目录存在且配置正确
3. **访问权限**：您需要能够访问 OpenClaw 的 Web 界面

## 集成步骤

### 步骤 1：访问 OpenClaw 定时任务页面

1. 打开浏览器，访问以下地址：
   ```
   http://127.0.0.1:18789/cron
   ```

2. 如果需要输入令牌，请使用以下令牌：
   ```
   2e47ede431dbd73de7ae4a2556ba69260427d1269e8878da
   ```

### 步骤 2：创建定时任务

1. 在定时任务页面，点击 "New Job"（新建任务）按钮

2. 填写以下配置信息：

   #### 基本信息
   - **Name**（名称）：`进化系统报告`
   - **Description**（描述）：`定期执行进化系统并生成进化报告`
   - **Agent ID**（智能体 ID）：`business`（谛听）
   - **Enabled**（已启用）：勾选

   #### 计划
   - **Schedule**（计划）：选择 `Every`（每隔）
   - **Every**（每隔）：设置为 `1`
   - **Unit**（单位）：选择 `hours`（小时）

   #### 执行
   - **Session**（会话）：选择 `isolated`（隔离）
   - **Wake mode**（唤醒模式）：选择 `now`（立即）
   - **What should run?**（执行什么？）：选择 `Run assistant task (isolated)`（运行助手任务（隔离））
   - **Prompt**（提示）：
     ```
     请执行进化系统并生成最新的进化报告，包含学习进度、获得的洞察、改进建议和下一步行动。

     执行命令：node evolver/index.js --loop
     ```

   #### 投递
   - **Result delivery**（结果投递）：选择 `Announce summary (default)`（通告摘要（默认））
   - **Channel**（频道）：选择 `feishu`（飞书）

3. 点击 "Add job"（添加任务）按钮保存配置

### 步骤 3：验证定时任务

1. 在定时任务列表中，您应该能看到刚刚创建的 "进化系统报告" 任务

2. 点击任务名称查看详情

3. 点击 "Trigger"（触发）按钮立即执行任务，验证任务是否正常工作

## 查看进化报告

1. 当定时任务执行时，进化系统会：
   - 连接到 EvoMap 进行学习
   - 生成学习日志并保存到 `evolver/learn-logs` 目录
   - 通过 OpenClaw 生成进化报告

2. 您可以通过以下方式查看进化报告：
   - 在 OpenClaw 定时任务页面查看任务执行历史
   - 查看飞书中的通知
   - 直接查看 `evolver/learn-logs` 目录中的学习日志文件

## 故障排除

### 常见问题

1. **OpenClaw 服务未运行**
   - 运行 `openclaw gateway start` 启动服务
   - 或运行 `启动OpenClaw.bat` 脚本

2. **定时任务执行失败**
   - 检查进化系统是否正确配置
   - 确保 `evolver` 目录存在且有执行权限
   - 检查 OpenClaw 日志以获取详细错误信息

3. **进化报告未生成**
   - 检查 `evolver/learn-logs` 目录是否有新的日志文件
   - 验证进化系统是否能够连接到 EvoMap

## 高级配置

### 调整执行频率

如果您想调整进化系统的执行频率，可以修改定时任务的计划设置：

- **每 30 分钟执行一次**：设置 `Every` 为 `30`，`Unit` 为 `minutes`
- **每 6 小时执行一次**：设置 `Every` 为 `6`，`Unit` 为 `hours`
- **每天执行一次**：设置 `Every` 为 `1`，`Unit` 为 `days`

### 自定义进化报告内容

您可以修改定时任务的提示内容，以自定义进化报告的格式和内容：

```
请执行进化系统并生成详细的进化报告，包括：
1. 学习进度和统计数据
2. 获得的关键洞察
3. 系统改进建议
4. 下一步行动计划
5. 学习日志分析

执行命令：node evolver/index.js --loop
```

## 总结

通过将进化系统与 OpenClaw 定时任务集成，您可以：

1. 定期自动执行进化系统
2. 通过 OpenClaw 界面查看进化报告
3. 接收飞书通知获取报告摘要
4. 监控进化系统的学习进度

这种集成方式确保您能够及时了解进化系统的状态和进展，为您的 AI 系统优化提供有价值的 insights。
