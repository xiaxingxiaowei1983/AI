# WaytoAGI 知识库

## 知识库结构

本知识库基于 WaytoAGI 社区的核心内容构建，旨在为 AI 学习者提供系统全面的学习资源和路径。

### 目录结构

- **core/**: WaytoAGI 核心内容和基础知识
- **learning-plans/**: 学习计划和路径
- **resources/**: 学习资源和工具
- **skills/**: 技能体系和实践指南
- **community/**: 社区活动和合作资源

## WaytoAGI 社区概览

WaytoAGI 是一个致力于人工智能学习的知识库与社区平台，提供系统全面的 AI 学习路径，整合资源、组织实践活动。

### 核心数据
- **覆盖学习者**：近 900 万
- **知识库访问量**：超 8000 万次
- **联动高校**：180 所
- **合作企业**：100+ 家
- **举办活动**：500+ 场
- **知识库内容**：10000+ 篇

### 核心功能
1. **系统全面的 AI 学习路径**
2. **丰富的学习资源整合**
3. **实践活动组织**
4. **高校和企业合作网络**
5. **飞书知识问答系统**
6. **公众号：通往 AGI 之路**

## 如何使用本知识库

1. **学习路径**：按照 `learning-plans` 中的学习计划进行系统学习
2. **资源查找**：在 `resources` 中查找所需的学习资源
3. **技能学习**：通过 `skills` 中的技能指南提升实践能力
4. **社区参与**：参考 `community` 中的活动信息参与社区活动
5. **内容贡献**：欢迎贡献新的学习资源和实践经验

## 知识库更新

本知识库将定期更新，确保内容与 WaytoAGI 社区的最新发展保持同步。
