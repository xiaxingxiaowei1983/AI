# 大脑作弊器 - 思维模型工具箱

全球顶级智慧 × 你的私人认知军火库。用100个可拆解的思维模型，帮你打破死局。

## 项目简介

"大脑作弊器"是一个思维模型工具箱，集成了：

1. **大脑作弊器**：不读废话，直取真经。将60分钟阅读压缩至5分钟行动指令
2. **思维模型库**：精选100个来自各领域的顶级思维模型
3. **4-Card系统**：独特的问题解决框架

## 核心功能

### 1. 认知脱水机模块

#### 全能输入
- **文件输入**：支持 PDF、EPUB、TXT 文件拖拽上传
- **链接输入**：支持网页链接，自动抓取内容摘要
- **截图输入**：支持图片上传，自动OCR识别文字

#### AI深度推演
- 基于"名道法术器例"六维模型：
  - **名**：核心概念 - 精确定义主题的本质
  - **道**：底层逻辑 - 揭示事物背后的根本规律
  - **法**：执行框架 - 提供系统化的行动方法论
  - **术**：具体方法 - 给出可直接执行的操作步骤
  - **器**：工具资源 - 推荐相关的工具、模板、书籍
  - **例**：实战案例 - 提供具体的应用场景

#### 一键导出
- **复制文字**：一键复制生成的脚本内容
- **下载文件**：将脚本保存为文本文件

### 2. 思维模型库

包含6个精选思维模型：
- 第一性原理
- 贝叶斯思维
- 决策树
- SWOT分析
- 5W2H分析法
- 波特五力模型

每个模型都包含详细的解释、应用步骤和实际案例。

### 3. 用户体验

- **响应式设计**：完美适配桌面和移动设备
- **流畅动画**：GSAP动画和滚动渐入效果
- **深色主题**：黑色极简设计
- **交互友好**：拖拽上传、一键操作

## 如何访问

### 开发环境

1. 确保项目已启动：
```bash
cd /workspace/projects
coze dev
```

2. 在浏览器中访问：
```
http://localhost:5000
```

### 访问认知脱水机

访问后，网站会自动跳转到主页面，向下滚动即可看到"认知脱水机"模块，或者点击导航栏的"认知脱水机"链接。

## 认知脱水机使用流程

1. **选择输入类型**
   - 点击"文件"、"链接"或"截图"按钮

2. **投喂内容**
   - 文件：拖拽文件到上传区域，或点击选择文件
   - 链接：粘贴网页链接
   - 截图：拖拽图片到上传区域，或点击选择图片

3. **生成脚本**
   - 点击"开始脱水"按钮
   - 等待30-60秒（AI深度推演中）

4. **查看结果**
   - 六段式卡片展示（名、道、法、术、器、例）
   - 每段配有专属图标和颜色标识

5. **导出保存**
   - 点击"复制"按钮复制到剪贴板
   - 点击"下载"按钮保存为文本文件
   - 点击"重新投喂"开始新的内容提取

## API接口

### POST /api/process

处理输入内容并生成认知脚本。

**请求体**：
```json
{
  "type": "file" | "url" | "image",
  "content": "文件base64数据 | URL字符串 | 图片base64数据"
}
```

**响应体**：
```json
{
  "title": "脚本标题",
  "name": "名的内容",
  "dao": "道的内容",
  "fa": "法的内容",
  "shu": "术的内容",
  "qi": "器的内容",
  "li": "例的内容"
}
```
