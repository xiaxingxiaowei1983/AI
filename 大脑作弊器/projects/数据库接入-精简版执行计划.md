## 🎯 用户体验

### 注册流程
1. 点击"登录"按钮
2. 点击"立即注册"
3. 填写用户名、邮箱、密码
4. 点击注册 → 成功后自动切换到登录界面

### 登录流程
1. 点击"登录"按钮
2. 输入用户名/邮箱 + 密码
3. 点击登录 → 成功后：
   - 保存 Token 到 localStorage
   - 更新登录按钮显示用户名
   - 显示成功提示

### 登出流程
- 点击登录按钮（显示用户名）
- 确认登出 → 清除 Token 并刷新页面

## 📊 数据库表结构

### users 表
| 字段 | 类型 | 说明 |
|------|------|------|
| id | varchar(36) | 主键（UUID） |
| username | varchar(128) | 用户名（唯一） |
| email | varchar(255) | 邮箱（唯一） |
| password | varchar(255) | 密码（bcrypt 加密） |
| role | varchar(20) | 角色：user/admin |
| isActive | boolean | 是否激活 |
| createdAt | timestamp | 创建时间 |
| updatedAt | timestamp | 更新时间 |

## 🔌 API 接口

### POST /api/auth/register
**请求**:
```json
{
  "username": "testuser",
  "email": "test@example.com",
  "password": "test123456"
}
```

**响应**:
```json
{
  "success": true,
  "message": "注册成功",
  "user": {
    "id": "uuid",
    "username": "testuser",
    "email": "test@example.com",
    "role": "user"
  }
}
```

### POST /api/auth/login
**请求**:
```json
{
  "username": "testuser",
  "password": "test123456"
}
```

**响应**:
```json
{
  "success": true,
  "message": "登录成功",
  "token": "jwt_token",
  "user": {
    "id": "uuid",
    "username": "testuser",
    "email": "test@example.com",
    "role": "user"
  }
}
```

### GET /api/auth/me
**请求头**:
```
Authorization: Bearer <token>
```

**响应**:
```json
{
  "success": true,
  "user": {
    "id": "uuid",
    "username": "testuser",
    "email": "test@example.com",
    "role": "user"
  }
}
```

## 💾 前端 API 使用

```javascript
// 注册
const result = await AuthAPI.register('username', 'email', 'password');

// 登录
const result = await AuthAPI.login('username', 'password');
AuthAPI.saveToken(result.token);

// 获取当前用户
const result = await AuthAPI.getCurrentUser();

// 检查是否已登录
if (AuthAPI.isLoggedIn()) {
    const user = AuthAPI.getUser();
}

// 登出
AuthAPI.logout();
```
