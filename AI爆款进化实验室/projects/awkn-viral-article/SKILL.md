# 公众号爆款文章一键生成

基于认知工程学和AWKN爆文逻辑，将书籍/文章拆解内容转化为公众号爆款文章

## 核心原则

- 不是内容写作技巧，而是一套认知工程学
- 外在结果只是内在认知的投射
- 不教读者"做什么"，而是重构"是谁"
- 所有行为改变的前提，是认知框架的改变

## 任务目标

- 本技能用于：基于书籍/文章拆解内容，一键生成公众号爆款文章
- 核心价值：从"普通内容"到"爆文"，实现认知冲突、情感共鸣、行动转化
- 能力包含：
  - 7大标题公式生成
  - 1500字标准版6模块正文创作
  - 转发文案和摘要生成
  - 完整的质量检查清单
- 触发条件：用户提供书籍/文章的拆解内容或原文

## 前置准备

- 已完成内容拆解（使用 awkn-content-decomposition 技能）
- 或提供完整的书籍/文章原文

## 操作步骤

### Phase 1：内容分析与提炼

从拆解报告中确定4个核心要素作为框架：

| 核心要素 | 说明 | 示例 |
|---------|------|------|
| 核心主题 | 1个关键词 | "知识内化" |
| 核心洞见 | 1句话本质 | "学习不是收集信息而是改变行为" |
| 目标受众 | 最需要的人 | "学习狂热但无行动改变的人" |
| 情绪痛点 | 最大的困惑/恐惧/渴望 | "付出时间金钱却原地踏步" |

**访问原文提取丰富素材**：
- 如果用户提供了文档文件：从原文中提取完整故事案例、更多金句（10-20条）、具体数据和引用、生动场景描述、情感化段落
- 如果用户提供了纯文本：直接从原文中提取上述素材
- 确保有充足的案例、数据和金句支撑文章观点

> **这步至关重要**。没有这4要素的精准提炼和丰富素材，后续所有创作都是空中楼阁。

### Phase 2：标题生成——制造认知冲突

使用7大公式生成标题（参考 [references/title-formulas.md](references/title-formulas.md)）：

| 公式类型 | 核心逻辑 | 示例 |
|---------|---------|------|
| 时间反差型 | 短期投入→长期回报 | "如何用30分钟，获得别人10年的洞察" |
| 身份锁定型 | 识别痛点+反直觉建议 | "如果你迷茫，接下来3个月别再学习新技能" |
| 反常识型 | 挑战主流认知 | "学习是个骗局（真正厉害的人从不上课）" |
| 技能价值型 | 制造稀缺感+紧迫感 | "未来10年最赚钱的能力，不是编程" |
| 问题揭露型 | 用问题引发自省 | "你为什么学了那么多却用不出来？答案可能让你震惊" |
| 直击痛点型 | 直接点名目标受众 | "如果你总是学习新东西却从不实践，这篇文章是写给你的" |
| 反向认同型 | 预设消极行为，激发反击欲 | "你大概率会读完这篇文章，然后什么都改变不了" |

**标题生成要点**：
- 标题不是概括，而是攻击，让读者不得不点开
- 结合原文中的对比案例生成更具冲击力的标题
- 提供3-5个标题选项供用户选择
- 这些公式可以叠加，效果加倍

### Phase 3：正文结构——6模块说服术

生成1500字标准版文章，精确到每个字的功能分配：

#### 钩子开场（前150字）

打破认知惯性，使用以下4种方式之一：
- **反向认同**：承认自己也曾犯错
- **自我揭露**：暴露弱点建立信任
- **反常识宣言**：直接挑战主流观点
- **直击痛点**：点名目标受众的困惑


```

#### 痛点共鸣（150-200字）

建立情感连接，包含3个要素：
- 承认自己也曾犯错（或引用原文中的具体场景）
- 列举无效方案（原文中的真实案例）
- 点明深层原因（揭示本质）

```

#### 价值承诺（100字）


```

#### 核心内容（800-1000字）

重构认知框架，3-5个模块，每个模块遵循以下结构：

1. **错误理解**：大多数人如何理解
2. **正确洞见**：揭示本质（用粗体标记）
3. **案例支撑**：原文中的具体案例和数据
4. **金句提炼**：用*斜体*标记，制造记忆点


```

#### 金句提炼（分布全文）

从原文中提取10-20条金句，选择3-5句最有力度的用*斜体*标记，其余灵活融入正文。

这些金句不是装饰品，而是社交货币：读者转发不是推荐文章，而是表达自己的认知升级。

#### 赋权结尾（150字）

激发行动/思考，提供明确的方向：

- **行动指南**：具体的行动建议
- **思考问题**：引发深度思考
- **认知升级**：完成身份重构


```

### Phase 4：语气控制——朋友式的叛逆者

#### 应该的语气

✅ 像朋友聊天，不像老师讲课
✅ 敢于说真话，有冲击力
✅ 深刻但接地气
✅ 承认局限，用具体例子
✅ 借鉴原文的叙述风格

#### 避免的语气

❌ 说教感：告诉读者"应该"做什么
❌ 空洞励志："相信自己，你一定行"
❌ 过度正能量：忽略现实的困难和挑战
❌ 抽象说教：没有具体例子的理论

### Phase 5：格式规范

严格遵循格式规范：

- 每段不超过3行（防止阅读疲劳）
- **粗体**：关键概念
- *斜体*：核心金句（至少3-5句）
- > 引用块：强化权威感（引用原文中的数据和引用）

### Phase 6：检查优化

确保文章符合以下标准：

✅ **金句数量**：至少3-5句可单独摘录的金句，且金句来自原文
✅ **反常识洞见**：有反常识洞见，打破认知惯性，且有原文案例支撑
✅ **情感共鸣**：建立了情感共鸣和信任连接（有具体故事和场景）
✅ **行动指南**：提供了可执行的行动指南，而非空洞建议
✅ **素材准确性**：所有素材（案例、数据、金句）都来自原文，保证准确性
✅ **字数控制**：1500字左右（±200字）

### Phase 7：转发文案生成——不被折叠的金句引导

生成1-2条转发文案（约140字内，不被微信折叠）：

**转发文案要点**：
- 使用金句或核心观点，精炼有冲击力
- 可加emoji增强视觉效果
- 吸引点击，提高打开率
- 格式：精炼短句 + 金句点缀


```

### Phase 8：摘要生成——100字核心概括

概括文章核心内容（100字内）：

**摘要要点**：
- 包含关键信息和价值点
- 吸引眼球，提高点击率
- 简洁明了，一语道破


```

## 使用说明

### 输入要求
- **首选**：提供书籍/文章的拆解报告（使用 awkn-content-decomposition 技能生成）
- **备选**：提供完整的书籍/文章原文或详细摘要
- **主题创作**：明确主题名称和关键内容点

### 创作流程
1. **内容分析**：从输入内容中提炼4个核心要素（主题、洞见、受众、痛点）
2. **素材提取**：提取金句、案例、数据等丰富素材（10-20条金句）
3. **标题生成**：基于7大标题公式，生成3-5个标题选项
4. **正文创作**：按6模块结构创作1500字正文（钩子→痛点→承诺→核心→金句→赋权）
5. **文案生成**：生成转发文案（140字内）和摘要（100字内）
6. **质量检查**：对照检查清单，确保质量和准确性

### 输出内容
- 标题选项（3-5个）
- 完整正文（1500字，含格式标记）
- 转发文案（1-2条）
- 摘要（1条）

### 核心要点
- 每次创作都必须基于原文的4个核心要素和丰富素材
- 切忌套用模板或参考特定示例的结构
- 保持多样性：不同主题应该有不同的切入点和风格
- 所有素材（案例、数据、金句）必须来自原文
