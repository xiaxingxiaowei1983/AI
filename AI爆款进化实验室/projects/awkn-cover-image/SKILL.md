# 文章封面生成器

为文章生成精美的手绘风格封面图，支持多种风格和尺寸。

## 使用方式

```bash
# 从Markdown文件（自动选择风格）
/awkn-cover-image path/to/article.md

# 指定风格
/awkn-cover-image path/to/article.md --style blueprint
/awkn-cover-image path/to/article.md --style warm
/awkn-cover-image path/to/article.md --style dark-atmospheric

# 不包含标题文字
/awkn-cover-image path/to/article.md --no-title

# 组合选项
/awkn-cover-image path/to/article.md --style minimal --no-title

# 直接输入内容
/awkn-cover-image
[粘贴内容或描述主题]

# 直接输入并指定风格
/awkn-cover-image --style playful
[粘贴内容]
```

## 选项

| 选项 | 说明 |
|------|------|
| `--style <name>` | 指定封面风格（见风格画廊） |
| `--aspect <ratio>` | 尺寸比例：2.35:1（电影感，默认）、16:9（宽屏）、1:1（方形社交） |
| `--lang <code>` | 标题文字的输出语言（zh、en、ja等） |
| `--no-title` | 生成不含标题文字的封面（仅视觉） |

## 工作流程

### 第1步：分析内容

1. **保存源内容**（如果不是文件）：
   - 用户提供文件路径：直接使用
   - 用户粘贴内容：保存到`source.md`

2. **提取关键信息**：
   - **主题**：文章关于什么？
   - **核心信息**：关键要点是什么？
   - **语气**：严肃、有趣、启发性、教育性？
   - **关键词**：识别风格信号词

3. **语言检测**：
   - 从内容检测**源语言**
   - 从对话检测**用户语言**
   - 注意源语言≠用户语言的情况（第3步会询问）

### 第2步：确定选项

1. **风格选择**：
   - 如果指定`--style`，使用该风格
   - 否则扫描内容寻找风格信号，自动推荐3个候选
   - 无明确信号时默认使用`elegant`

2. **尺寸比例**：
   - 如果指定`--aspect`，使用该比例
   - 否则准备选项：2.35:1（电影感）、16:9（宽屏）、1:1（方形）

### 第3步：确认选项

**目的**：生成前让用户一次性确认所有选项。

**重要**：使用AskUserQuestion一次性展示所有选项，不要多次打断。

**确定需要询问的问题**：

| 问题 | 何时询问 |
|------|---------|
| 风格 | 总是（必需） |
| 尺寸 | 总是（提供常用选项） |
| 语言 | 仅当`源语言≠用户语言`时 |

**展示选项**（使用AskUserQuestion包含所有适用问题）：

**问题1（风格）** - 总是：
- 风格A（推荐）：[风格名称] - [简要描述]
- 风格B：[风格名称] - [简要描述]
- 风格C：[风格名称] - [简要描述]
- 自定义：提供自定义风格参考

**问题2（尺寸）** - 总是：
- 2.35:1 电影感（推荐）- 超宽、戏剧性
- 16:9 宽屏 - 标准视频/演示
- 1:1 方形 - 社交媒体优化

**问题3（语言）** - 仅当源语言≠用户语言时：
- [源语言]（匹配内容）
- [用户语言]（你的偏好）

**语言处理**：
- 源语言=用户语言：直接告知用户（如："标题将是中文"）
- 源语言≠用户语言：询问使用哪种语言作为标题文字

### 第4步：生成封面概念

根据选定的风格创建封面图像概念：

**标题**（如果包含，最多8字符）：
- 将核心信息提炼为有力的标题
- 使用钩子：数字、问题、对比、痛点
- 如果使用`--no-title`标志则跳过

**视觉元素**：
- 符合风格的图像和图标
- 1-2个代表主题的象征性元素
- 符合风格的隐喻或类比

### 第5步：生成图像

**图像生成技能选择**：
1. 检查可用的图像生成技能
2. 如果有多个技能，询问用户选择

**生成**：
调用选定的图像生成技能，使用提示词文件、输出路径和确认的尺寸比例。

### 第6步：输出总结

```
封面图已生成！

主题：[主题]
风格：[风格名称]
尺寸：[尺寸比例]
标题：[封面标题]（或"无标题 - 仅视觉"）
语言：[确认的语言]
位置：[输出路径]

预览图像以验证是否符合您的期望。
```

## 注意事项

- 封面在小预览尺寸下应立即理解
- 标题（如果包含）必须可读且有冲击力
- 视觉隐喻比字面表示更有效
- 保持整个封面的风格一致性
- 图像生成通常需要10-30秒
- 标题文字使用用户确认的语言偏好
- 尺寸比例：2.35:1用于电影感/戏剧性，16:9用于宽屏，1:1用于社交媒体
