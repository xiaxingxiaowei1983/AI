## 设计理念

网页不是随意设计的，它是一套**引导注意力的视觉层次系统**。

核心原则：
- **排版即认知**：视觉设计直接影响读者如何处理信息
- **降低认知负荷**：通过层次化设计让信息更容易被吸收
- **引导注意力**：用视觉元素引导读者的阅读路径

---

## 视觉层次系统

### 层次结构

| 视觉元素 | 作用 | 心理学原理 | 实现方式 |
|----------|------|------------|----------|
| **120px 空心标题字** | 第一眼吸引注意 | 大尺寸+高对比度 = 视觉捕捉 | font-size: 120px; font-weight: 300; |
| **绿色背景的关键词** | 突出核心概念 | 颜色对比 = 信息分层 | background: #00ff00; color: #000; |
| **倾斜5°的短语** | 打破视觉单调 | 非对称设计 = 认知唤醒 | transform: rotate(5deg); |
| **40px 行高的正文** | 降低阅读疲劳 | 黄金行距（1.6-1.8）= 舒适阅读 | line-height: 1.8; |
| **段落间距60px** | 创造认知呼吸空间 | 留白 = 降低认知负荷 | margin-bottom: 60px; |

### 视觉引导路径

```
1. 封面图（视觉钩子）
   ↓
2. 120px 空心标题（第一注意点）
   ↓
3. 绿色关键词（核心概念）
   ↓
4. 正文内容（信息吸收）
   ↓
5. 斜体金句（记忆锚点）
   ↓
6. 行动指南（行为触发）
```

## 留白艺术

### 留白的价值

AWKN 的页面不是充满内容，而是精心设计的空白：

1. **段落间距60px**：创造认知呼吸空间
   - 让大脑有时间处理信息
   - 降低认知负荷
   - 提高理解度

2. **左侧边栏目录**：提供认知锚点
   - 帮助读者建立内容框架
   - 快速定位关键信息
   - 降低迷失感

3. **80%的文字留白**：让观点"呼吸"
   - 避免信息过载
   - 提高可读性
   - 增强视觉舒适度

### 留白心理学

**认知呼吸理论**：
- 大脑需要时间处理信息
- 留白提供"认知缓冲区"
- 过多的信息会导致认知过载

**实现原则**：
- 每个模块之间至少60px间距
- 段落之间至少30px间距
- 页面边缘至少40px边距

## 颜色心理学

### 极简配色

**黑白主色调**：
- 黑色背景：最大程度突出文字，减少视觉干扰
- 白色文字：经典高对比度，符合阅读习惯

**单一强调色**：
- #00ff00 绿色：代表"希望/成长/行动"
- 用于：关键词、金句、重要信息

### 颜色功能

| 颜色 | 作用 | 心理效应 | 使用场景 |
|------|------|----------|----------|
| **黑色（#000000）** | 背景 | 消除干扰，突出内容 | 整体背景 |
| **白色（#FFFFFF）** | 正文 | 高对比度，易于阅读 | 所有文字 |
| **绿色（#00ff00）** | 强调 | 希望、成长、行动 | 关键词、金句 |
| **灰色（#666666）** | 次要信息 | 降低视觉权重 | 日期、注释 |

### 禁止的颜色

- ❌ 彩色渐变
- ❌ 多种强调色混用
- ❌ 霓虹效果
- ❌ 复杂的背景图案

## 排版规范

### 字体层级

```css
/* 一级标题：120px 空心字 */
h1 {
    font-size: 120px;
    font-weight: 300;
    letter-spacing: -0.02em;
}

/* 二级标题：48px */
h2 {
    font-size: 48px;
    font-weight: 600;
    letter-spacing: -0.01em;
}

/* 三级标题：32px */
h3 {
    font-size: 32px;
    font-weight: 600;
}

/* 正文：18px */
body {
    font-size: 18px;
    line-height: 1.8;
}
```

### 行间距

| 元素 | 行间距 | 原理 |
|------|--------|------|
| 标题 | 1.2 | 紧凑，突出重点 |
| 正文 | 1.6-1.8 | 黄金比例，舒适阅读 |
| 引用 | 1.6 | 稍松，突出引用 |

### 段落间距

| 场景 | 间距 | 原理 |
|------|------|------|
| 段落之间 | 30px | 认知呼吸 |
| 模块之间 | 60px | 明确分隔 |
| 章节之间 | 80px | 强烈分隔 |

## 响应式设计

### 断点设置

```css
/* 桌面端 > 1024px */
.container {
    max-width: 960px;
    padding: 0 40px;
}

/* 平板端 768px - 1024px */
@media (max-width: 1024px) {
    .container {
        max-width: 720px;
        padding: 0 30px;
    }
    h1 {
        font-size: 96px;
    }
}

/* 移动端 < 768px */
@media (max-width: 768px) {
    .container {
        max-width: 100%;
        padding: 0 20px;
    }
    h1 {
        font-size: 64px;
    }
    body {
        font-size: 16px;
    }
}
```

### 移动端优化

1. **标题字号调整**：
   - 桌面：120px
   - 平板：96px
   - 手机：64px

2. **行间距调整**：
   - 桌面：1.8
   - 移动：1.6（空间更紧凑）

3. **间距调整**：
   - 桌面：60px
   - 移动：40px（节省空间）

4. **导航简化**：
   - 桌面：完整导航栏
   - 移动：简化菜单

## 注意事项

### ✅ 必须遵循

- 黑白主色调
- 单一强调色（#00ff00）
- 大量留白（80%文字留白）
- 清晰的视觉层次
- 响应式设计

### ❌ 避免

- 多种颜色混用
- 过度的装饰元素
- 紧凑的排版
- 复杂的背景图案
- 非响应式设计

---

## 总结

**网页设计即认知工程**：

- 视觉层次引导注意力
- 留白创造认知呼吸空间
- 颜色传递情感和信息
- 排版降低认知负荷
- 响应式确保可访问性

记住：**排版即认知，设计即工程**。
