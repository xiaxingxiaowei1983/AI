# 人生决策宗师 - 产品路线图 (Roadmap)

## 目标平台
- 首发：微信小程序（主端）
- 管理后台：Web Admin（支持管理员登录、改密、内容运营）
- 后续：iOS/Android App（复用后端能力）

## 视觉与尺寸规范（微信端）
- 设计基准：`375 x 812`（iPhone X 逻辑分辨率）
- 适配策略：使用 `rpx` + 安全区（status bar / home indicator）
- 页面结构：暗色极简、黑金风，卡片折叠式信息层级
- 参考风格：`https://life-choice.vercel.app/`

## Phase 1 - v1.0 (MVP，1~2个月)
### 核心目标
跑通「身份建立 -> 早中晚循环 -> 基础推演 -> 积分转化」闭环。

### P0 功能清单
1. **Onboarding + 记忆档案**
   - 输入：姓名(可选)、性别、生日、时辰(未知可选)、出生地
   - 自动记忆用户档案（再次进入无需重复填写）
   - 生成基础命盘与置信度标识
2. **注册与积分**
   - 邮箱注册成功赠送 10 积分
   - 首日自动解锁早/中/晚体验
3. **日常飞轮（早/中/晚）**
   - 早：醒神帖（点醒今日需防的人/事）
   - 中：破局锦囊（一个可执行动作）
   - 晚：胜率复盘（文本/语音，沉淀上下文）
4. **宗师推演（对话化）**
   - 从“固定四维模块”改为“自由对话式推演”
   - 保留历史会话记录（可回溯）
5. **后台管理（Admin）**
   - 登录入口
   - 初始管理员：`10919669@qq.com`
   - 支持管理员自主修改密码
6. **偏好设置 & 多语言**
   - 中文 / English / 日本語
   - 文案可配置
7. **文案安全合规**
   - 关键词过滤（本地词库 + 第三方内容安全）

### 关键验收
- 早中晚接口响应 < 500ms
- 宗师推演首字返回 < 2s（流式）
- 邮箱注册->积分到账成功率 > 99%

## Phase 2 - v1.5（1个月）
### 核心目标
构建会员与积分护城河，提升复盘留存。

### P0 功能清单
1. 会员体系（Free/Basic/Premium）
2. 积分商城与充值
3. 晚盘 AI 反馈精细化（偏误标签、策略建议）
4. 海报分享裂变（邀请双方得积分）
5. “宗师对您的理解程度”进度可视化

## Phase 3 - v2.0（2个月）
### 核心目标
从个人决策外脑升级为关系型决策网络。

### P0 功能清单
1. 预约宗师沟通（高级会员按次付费）
2. 关系协同推演（合伙/家庭/团队）
3. 宗师月报/季报
4. 企业微信与私域联动
