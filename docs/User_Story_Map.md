# 人生决策宗师 - 用户故事地图 (User Story Map)

## 横向活动流（Backbone）
1. 身份建立（Onboarding）
2. 每日决策（Daily Loop）
3. 重大抉择（Grandmaster）
4. 成长反馈（Feedback）
5. 支付与会员（Monetization）
6. 设置与后台（Settings/Admin）

---

## 1) 身份建立（Onboarding）
### P0
- 作为新用户，我希望输入出生信息并完成时空校准，建立专属档案。
- 作为新用户，我希望系统记住我的信息，下次打开可直接进入首页。

### P1
- 作为用户，我希望补充行业、职业阶段，提升建议准确度。

## 2) 每日决策（Daily Loop）
### P0
- 作为用户，我希望早晨看到「醒神帖」，快速知道今天应防范的人/事。
- 作为用户，我希望中午打开「破局锦囊」，得到一个可立即执行的动作。
- 作为用户，我希望晚上填写「胜率复盘」，让系统越用越懂我。

### P1
- 作为用户，我希望语音输入复盘，减少打字成本。
- 作为用户，我希望分享今日金句海报给朋友。

## 3) 重大抉择（宗师推演）
### P0
- 作为用户，我希望通过自由对话进行宗师推演，而不是被固定模板限制。
- 作为用户，我希望保留历史对话，方便复盘决策路径。

### P1
- 作为高级会员，我希望预约真人宗师沟通，处理关键节点决策。

## 4) 成长反馈（Feedback）
### P0
- 作为用户，我希望看到「宗师对您的理解程度」，感知系统进化。
- 作为用户，我希望查看本月高光日历，提前规避风险时段。

## 5) 支付与会员（Monetization）
### P0
- 作为注册用户，我希望邮箱注册后立即获得 10 积分。
- 作为用户，我希望可按积分/会员方式解锁功能。

### P1
- 作为用户，我希望看到单次付费与升级会员的性价比对比。

## 6) 设置与后台（Settings/Admin）
### P0
- 作为用户，我希望在偏好设置中切换中/英/日语言。
- 作为管理员，我希望通过后台入口登录并自主修改密码。
