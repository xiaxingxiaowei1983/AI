## 结构概览

AWKN 的文章像代码一样模块化，每个模块一个主题，每个主题一个洞见。

| 模块 | 字数 | 功能 | 写作技巧 |
|------|------|------|----------|
| 钩子开场 | 前150字 | 打破认知惯性 | 反向认同/自我揭露/反常识宣言/直击痛点 |
| 痛点共鸣 | 150-200字 | 建立情感连接 | 承认自己也曾犯错+列举无效方案+点明深层原因 |
| 价值承诺 | 100字 | 降低阅读门槛 | 明确将提供的具体价值，避免空洞承诺 |
| 核心内容 | 800-1000字 | 重构认知框架 | 3-5个模块，每个包含：错误理解→正确理解→案例→金句 |
| 金句提炼 | 分布全文 | 制造传播点 | 至少3-5句可单独摘录的金句，用斜体标记 |
| 赋权结尾 | 150字 | 激发行动/思考 | 行动指南/思考问题/认知升级 |

**总字数**：约1500字

## 模块1：钩子开场

### 功能
打破认知惯性，让读者不得不继续阅读

### 字数
前150字

### 写作技巧

#### 技巧1：反向认同
承认读者的行为是正常的，然后给出反常识结论

**示例：**
```
你每天都在努力学习，这很好。但让我告诉你一个残酷的真相：学习可能是你最浪费时间的行为。
```

#### 技巧2：自我揭露
承认自己也曾犯错，建立信任

**示例：**
```
我花了十年时间做错了所有事情，但这正是我能给你建议的原因。我曾经和你一样，以为只要努力就能成功。
```

#### 技巧3：反常识宣言
直接抛出颠覆性观点

**示例：**
```
学习是个骗局。真正厉害的人从不上课。
```

#### 技巧4：直击痛点
直接点出读者的核心困惑

**示例：**
```
你为什么学了那么多却用不出来？这个问题困扰了我三年，直到我发现真相。
```

### 注意事项
- 必须在前150字内完成
- 必须打破认知惯性
- 避免空洞的励志表达

## 模块2：痛点共鸣

### 功能
建立情感连接，让读者感觉"这就是在说我"

### 字数
150-200字

### 写作结构

#### 第1步：承认自己也曾犯错
```
我曾经和你一样，每天学习新知识，参加各种课程，收集大量资料。我以为这就是成长。
```

#### 第2步：列举无效方案
```
我尝试过番茄工作法，试过GTD，买过无数课程。但结果呢？我只是变得更焦虑了。
```

#### 第3步：点明深层原因
```
问题的根源不是我不够努力，而是我搞错了方向。我把"知道"当成了"做到"。
```

### 注意事项
- 具体描述，避免抽象
- 必须是目标受众的真实体验
- 过渡要自然，引出价值承诺

## 模块3：价值承诺

### 功能
降低阅读门槛，明确读者将获得什么

### 字数
100字

### 写作原则

#### 原则1：具体化
避免空洞承诺，明确提供具体价值

**❌ 错误示例：**
```
这篇文章会让你受益匪浅。
```

**✅ 正确示例：**
```
接下来5分钟，我会告诉你一个简单的方法，让你从今天开始，把学到的任何知识都变成实际行动。
```

#### 原则2：时间承诺
明确阅读时间，降低心理门槛

**示例：**
```
这篇文章需要你5分钟阅读，但会帮你节省未来5年的弯路。
```

### 注意事项
- 承诺必须可实现
- 避免过度营销化
- 与核心内容保持一致

## 模块4：核心内容

### 功能
重构认知框架，提供深度洞见

### 字数
800-1000字

### 写作结构

#### 核心内容 = 3-5个子模块

每个子模块包含4个要素：
1. 错误理解
2. 正确理解
3. 案例/证据
4. 金句

#### 子模块示例

**模块1：错误理解 vs 正确理解**
```
大多数人认为学习就是收集信息。你读完一本书，参加一个课程，就觉得学到了东西。

但这是错误的。真正的学习不是收集信息，而是改变行为。

我有个朋友，每年读100本书，但他的人生没有任何改变。而另一个朋友，一年只读3本书，但把书中的每个方法都付诸实践，现在年入百万。

*学习不是你获得了什么，而是你变成了什么。*
```

**模块2：错误理解 vs 正确理解**
```
大多数人认为努力工作就能成功。你每天工作12小时，觉得自己很勤奋。

但努力不是成功的决定因素，选择才是。

巴菲特说："我成功的秘诀，不是努力工作，而是选择正确的事情去努力。"

*成功来自于选择正确的事情去努力，而非努力工作。*
```

**模块3：错误理解 vs 正确理解**
```
大多数人认为坚持就是美德。你告诉自己要再坚持一下，不能放弃。

但坚持不一定是美德，有时候放弃才是智慧。

乔布斯在大学退学，因为他发现所学不是他想要的。正是这次放弃，让他走上了改变世界的道路。

*聪明人知道何时放弃，聪明人也知道何时坚持。*
```

### 注意事项
- 每个子模块聚焦一个核心洞见
- 必须有具体案例支撑
- 每个模块以金句结尾
- 保持逻辑连贯

## 模块5：金句提炼

### 功能
制造传播点，让读者可以摘录分享

### 分布
全文至少3-5句，分布在各个模块结尾

### 金句特征

#### 特征1：简洁有力
一句话说清楚一个洞见

**示例：**
```
学习不是你获得了什么，而是你变成了什么。
成功来自于选择正确的事情去努力，而非努力工作。
```

#### 特征2：反常识
打破认知惯性

**示例：**
```
学习是个骗局。
努力工作是个陷阱。
```

#### 特征3：可独立摘录
脱离上下文也能理解

**示例：**
```
*大多数人认为成功来自于努力工作。真相是，成功来自于选择正确的事情去努力。*
```

### 格式规范
使用斜体标记：`*金句内容*`

### 注意事项
- 至少3-5句金句
- 分布在全文，不要集中在结尾
- 确保金句能脱离上下文独立理解

## 模块6：赋权结尾

### 功能
激发行动或思考，让读者有收获感

### 字数
150字

### 写作技巧

#### 技巧1：行动指南
给出具体可执行的建议

**示例：**
```
从今天开始，停止学习新技能。找出你已经知道的5个技能，每个技能练习30天。这就是全部。
```

#### 技巧2：思考问题
引导读者深度思考

**示例：**
```
问问自己：你现在最需要学习的不是什么新技能，而是什么你已经知道但还没做到的事？
```

#### 技巧3：认知升级
总结认知层面的转变

**示例：**
``你现在明白为什么学了很多却用不出来。不是因为你不聪明，而是因为你搞错了方向。改变方向，比改变行为更重要。*

从今天开始，重新定义你是谁。
```

### 注意事项
- 必须给出具体可执行的建议
- 避免空洞的励志表达
- 与核心内容保持一致

## 完整示例

### 标题：学习是个骗局（真正厉害的人从不上课）

#### 钩子开场（150字）
你每天都在努力学习，这很好。但让我告诉你一个残酷的真相：学习可能是你最浪费时间的行为。

#### 痛点共鸣（200字）
我曾经和你一样，每天学习新知识，参加各种课程，收集大量资料。我以为这就是成长。

我尝试过番茄工作法，试过GTD，买过无数课程。但结果呢？我只是变得更焦虑了。我投入的时间和金钱，几乎没有带来任何实质性改变。

问题的根源不是我不够努力，而是我搞错了方向。我把"知道"当成了"做到"。

#### 价值承诺（100字）
接下来5分钟，我会告诉你一个简单的方法，让你从今天开始，把学到的任何知识都变成实际行动。这不是什么高深理论，而是一个你明天就能用上的具体方法。

#### 核心内容（1000字）

**模块1：学习 vs 知识内化**
大多数人认为学习就是收集信息。你读完一本书，参加一个课程，就觉得学到了东西。

但这是错误的。真正的学习不是收集信息，而是改变行为。

我有个朋友，每年读100本书，但他的人生没有任何改变。而另一个朋友，一年只读3本书，但把书中的每个方法都付诸实践，现在年入百万。

*学习不是你获得了什么，而是你变成了什么。*

**模块2：收集 vs 践行**
大多数人认为知道就是做到了。你听了一个方法，觉得自己已经掌握了。

但知道和做到之间，隔着巨大的鸿沟。

我自己就是个例子。我知道番茄工作法，但从来没用过。我知道GTD，但从来不用。我知道无数方法，但没有任何改变。直到有一天，我强迫自己每天只用一个方法，坚持30天。那30天改变了我的一切。

*知道不是力量，做到才是。*

**模块3：广度 vs 深度**
大多数人认为学得越多越好。你不断学习新技能，觉得自己很厉害。

但深度远比广度重要。

查理·芒格说："在重要的领域，成为前10%的专家，比什么都略懂一点强100倍。"

与其学习10个新技能，不如把1个技能练到极致。

*深度是唯一能产生复利的行为。*

#### 赋权结尾（150字）
从今天开始，停止学习新技能。

找出你已经知道的1个技能，这个技能你有70%的掌握度。然后，花30天，每天30分钟，把这个技能练到95%的掌握度。

这就是全部。没有复杂的理论，没有繁琐的步骤。

30天后，你会发现，你的能力提升不是30%，而是300%。

因为你终于明白了：*成功来自于选择正确的事情去努力，而非努力工作。*

## 格式规范总结

- 每段不超过3行（防止阅读疲劳）
- 粗体：关键概念
- 斜体：核心金句
- 引用块：强化权威感
- 使用表格展示结构
- 保持段落清晰，层次分明
