# 认知工程学：从认知重构到行动改变的完整闭环

## 任务目标

本 Skill 是一套认知工程学系统，不是教你"如何写"，而是教你如何重新连接读者的认知神经。核心能力包括：

- **神经钩子系统**：通过4种神经钩子激活视觉皮层、前额叶皮层、边缘系统、海马体
- **认知冲突设计**：打破原有框架，制造认知惯性破裂
- **模块化架构**：可插拔的认知模块，符合7±2工作记忆限制
- **视觉层次系统**：引导注意力的排版设计，降低认知负荷
- **公式化写作**：可复制的创作机器，精确到字数控制
- **传播工程学**：设计分享动机，创造社交货币

触发条件：用户需要创作观点类、方法论类、认知类深度内容，且希望内容具备高传播性和认知冲击力。

## 核心理念

**这不是写作技巧，而是认知工程学**

本系统的本质是用语言作为手术刀，精准切除读者思维中的错误认知，植入新的框架。通过神经钩子激活不同脑区，用模块化架构降低认知负荷，用视觉层次引导注意力，用传播工程学设计分享动机。

**完整的认知到行为闭环**：
认知冲突 → 情感共鸣 → 洞见解构 → 知识输入 → 记忆强化 → 行动指南 → 认知升级

**身份优先于行动**：外在结果只是内在认知的投射。所有行为改变的前提，是认知框架的改变。不教"做什么"，而是重构"是谁"。

## 神经钩子系统

### 4种神经钩子及其作用

| 钩子类型 | 激活脑区 | 作用阶段 | 功能 |
|----------|----------|----------|------|
| **视觉钩子** | 视觉皮层 | Phase 4（封面图） | 创造第一印象，第一眼吸引注意 |
| **认知钩子** | 前额叶皮层 | Phase 2（标题） | 激发逻辑思考，制造认知冲突 |
| **情感钩子** | 边缘系统 | Phase 3（钩子开场） | 建立情感连接，引发共鸣 |
| **记忆钩子** | 海马体 | 全文（斜体金句） | 创造记忆锚点，便于传播 |

### 神经激活链条

完整的神经激活必须按以下顺序：

```
1. 视觉钩子（Phase 4） → 激活视觉皮层 → 创造第一印象
   ↓
2. 认知钩子（Phase 2） → 激活前额叶皮层 → 打破认知惯性
   ↓
3. 情感钩子（Phase 3） → 激活边缘系统 → 建立情感连接
   ↓
4. 记忆钩子（全文） → 激活海马体 → 创造记忆锚点
```

**设计原则**：4种钩子必须协同作用，缺一不可。

## 操作步骤

### Phase 1：内容分析与提炼

接收用户的想法、观点、素材或灵感碎片，提炼出4个核心要素：

**智能体执行以下分析：**

1. **核心主题**：1个关键词
   - 示例："知识内化"、"认知升级"、"行动力"

2. **核心洞见**：1句话本质
   - 要求：必须是反常识的、能打破认知惯性的
   - 示例："学习不是收集信息而是改变行为"
   - 示例："成功来自于选择正确的事情去努力，而非努力工作"

3. **目标受众**：最需要这个洞见的人
   - 示例："学习狂热但无行动改变的人"
   - 示例："每天工作超过10小时却无实质性进展的人"

4. **情绪痛点**：最大的困惑/恐惧/渴望
   - 示例："付出时间金钱却原地踏步"
   - 示例："努力了却看不到回报"

**完成标志**：4要素清晰明确，核心洞见具备反常识特性。

---

### Phase 2：标题生成——制造认知冲突

使用7大标题公式生成3-5个备选标题。详细公式说明见 [references/title-formulas.md](references/title-formulas.md)

**智能体执行以下操作：**

基于 Phase 1 的4要素，使用以下公式生成标题（建议生成3-5个供选择）：

1. **时间反差型**：短期投入→长期回报
   - "如何用30分钟，获得别人10年的洞察"

2. **身份锁定型**：识别痛点+反直觉建议
   - "如果你迷茫，接下来3个月别再学习新技能"

3. **反常识型**：挑战主流认知
   - "学习是个骗局（真正厉害的人从不上课）"

4. **技能价值型**：制造稀缺感+紧迫感
   - "未来10年最赚钱的能力，不是编程"

5. **问题揭露型**：用问题引发自省
   - "你为什么学了那么多却用不出来？答案可能让你震惊"

6. **直击痛点型**：直接点名目标受众
   - "如果你总是学习新东西却从不实践，这篇文章是写给你的"

7. **反向认同型**：预设消极行为，激发反击欲
   - "你大概率会读完这篇文章，然后什么都改变不了"

**组合策略**：公式可以叠加，效果加倍
- 示例："学习是个骗局。如果你每天工作超过10小时，接下来1个月别再努力工作。"

**完成标志**：生成3-5个标题，每个标题都具备认知冲突特性。

---

### Phase 3：正文结构——6模块说服术

按照1500字标准版写作，每个模块有明确字数和功能分配。详细模板见 [references/content-structure.md](references/content-structure.md)

**智能体执行以下写作：**

使用以下6模块结构，严格控制在1500字左右：

| 模块 | 字数 | 功能 | 写作技巧 |
|------|------|------|----------|
| **钩子开场** | 前150字 | 打破认知惯性 | 反向认同/自我揭露/反常识宣言/直击痛点 |
| **痛点共鸣** | 150-200字 | 建立情感连接 | 承认自己也曾犯错+列举无效方案+点明深层原因 |
| **价值承诺** | 100字 | 降低阅读门槛 | 明确将提供的具体价值，避免空洞承诺 |
| **核心内容** | 800-1000字 | 重构认知框架 | 3-5个模块，每个包含：错误理解→正确理解→案例→金句 |
| **金句提炼** | 分布全文 | 制造传播点 | 至少3-5句可单独摘录的金句，用斜体标记 |
| **赋权结尾** | 150字 | 激发行动/思考 | 行动指南/思考问题/认知升级 |

**格式规范：**
- 每段不超过3行（防止阅读疲劳）
- 粗体：关键概念
- 斜体：核心金句
- 引用块：强化权威感

**语气控制**：详细说明见 [references/tone-control.md](references/tone-control.md)

**完成标志**：完成1500字正文，符合6模块结构，至少3-5句金句。

### Phase 4：视觉隐喻——超现实木刻风格

使用图像生成能力创建封面图，严格遵循维多利亚木刻/蚀刻风格。详细规范见 [references/visual-metaphor.md](references/visual-metaphor.md)

**智能体执行以下操作：**

1. **理解核心洞见**：从 Phase 1 的4要素中提取视觉化元素
2. **生成图像描述**：遵循以下设计原则：
   - 纯黑白，16:9横向构图
   - 预留纯黑文字区域
   - 超现实转化（如：装满书的头颅，但书页全是空白）
   - 横排线条+点刻质感

3. **调用图像生成能力**：
   - 根据描述生成符合要求的图像
   - 确保风格：维多利亚木刻/蚀刻风格、超现实、黑白、16:9

**禁令：**
- 禁止：彩色/渐变/3D/卡通
- 禁止：模糊/光晕/滤镜
- 禁止：字面翻译（直接画书本是失败案例）
- 禁止：超过3个主要元素

**完成标志**：生成一张符合超现实木刻风格的封面图。

### Phase 5：极简网页——格栅主义

生成完整的网页代码，采用极简格栅主义风格。模板示例见 [assets/webpage-template.md](assets/webpage-template.md)

**智能体执行以下操作：**

生成一个完整的HTML/CSS网页，遵循以下设计原则：

1. **视觉风格：**
   - 黑白主色调+单一强调色（默认#00ff00绿色）
   - 几何装饰元素
   - 响应式布局
   - 预留文字区域

2. **布局结构：**
   - 简洁的导航栏
   - 封面图区域（16:9比例）
   - 标题区域
   - 正文内容区域（分模块展示）
   - 底部区域

3. **代码规范：**
   - 使用语义化HTML标签
   - CSS使用Flexbox/Grid布局
   - 确保响应式适配（移动端友好）
   - 代码整洁，易于修改

**完成标志**：生成一个可独立运行的HTML文件。

## 完整认知闭环

从认知到行为的系统工程，7个阶段：

1. **认知冲突**：打破原有框架
   - 不教你"如何更好地学习"，而是告诉你"学习是个骗局"
   - 先打破读者的认知框架，再植入新的

2. **情感共鸣**：建立信任连接
   - 承认自己也曾犯错："我曾是一个典型的学习狂热者..."
   - 让读者觉得"他懂我"

3. **洞见解构**：重构认知模型
   - 给出反常识洞见："学习不是收集信息，而是改变行为"
   - 重构读者理解世界的方式

4. **知识输入**：模块化信息处理
   - 3-5个模块，每个模块遵循"错误理解→正确洞见→案例→金句"
   - 确保信息能被读者吸收（符合7±2工作记忆限制）

5. **记忆强化**：创造可提取的认知锚点
   - 用斜体金句创造记忆点："收集信息不是学习，改变行为才是"
   - 让读者在需要时能轻松提取

6. **行动指南**：从认知到行为的转化
   - 给出明确的行动指令："从今天开始，停止学习新东西。选择一个你已经知道的概念，深度实践它30天。"
   - 避免空洞的励志表达

7. **认知升级**：完成身份重构
   - 最终输出："学习不是收集信息，而是改变行为。身份不是由你做什么决定的，而是由你成为谁决定的。"
   - 完成从"学习者"到"实践者"的身份重构

**关键原则**：
- 读者转发不是推荐文章，而是表达自己的认知升级
- 金句是社交货币，必须可单独摘录
- 诚实是最好的营销，错误是信任的来源

## 高级技巧

### 反常识洞见的寻找方法

1. **逆流行**：主流观点的对立面
   - "停止学习新技能" vs "终身学习"
   - "努力工作是个陷阱" vs "勤奋是美德"

2. **寻本质**：现象背后的底层逻辑
   - "学习不是收集信息而是改变行为"
   - "行动力不是动力而是选择的能力"

3. **跨学科**：其他领域的原理迁移
   - 从认知科学到写作
   - 从编程到行动力设计

### 模块化架构原则

每个模块控制在4-5个信息点，符合7±2工作记忆限制：

```
## [模块标题：点出核心洞见]
> "引用金句" —— 出处

[先说错误理解]
但真相是：[给出正确理解]
举个例子：[具体案例]

*金句提炼*
```

**结构优势**：
- 可插拔：每个模块独立，可灵活组合
- 可复制：相同的结构确保输出质量稳定
- 可记忆：标准化结构降低认知负荷

### 视觉层次系统

引导注意力的排版设计，降低认知负荷：

| 视觉元素 | 作用 | 心理学原理 |
|----------|------|------------|
| 120px 空心标题字 | 第一眼吸引注意 | 大尺寸+高对比度 = 视觉捕捉 |
| 绿色背景的关键词 | 突出核心概念 | 颜色对比 = 信息分层 |
| 倾斜5°的短语 | 打破视觉单调 | 非对称设计 = 认知唤醒 |
| 40px 行高的正文 | 降低阅读疲劳 | 黄金行距（1.6-1.8）= 舒适阅读 |
| 段落间距60px | 创造认知呼吸空间 | 留白 = 降低认知负荷 |

## 最终洞见

本 Skill 不是写作技巧集合，而是认知工程学系统。

**核心区别**：
- 写作技巧关注"如何表达"
- 认知工程学关注"如何重构认知"

**本质区别**：
- 写作技巧是"术"，教你把话说漂亮
- 认知工程学是"道"，教你用语言作为手术刀

**最终目标**：
不是让读者觉得"这篇文章写得真好"，而是让读者觉得"我以前想错了"。

当读者说"我以前想错了"的那一刻，认知重构完成，行为改变开始。

## 传播工程学

### 社交货币设计

读者转发你的文章，不是为了推荐内容，而是为了表达自己的认知升级。

**社交货币的本质**：
- 读者通过转发文章，向社交圈展示："我升级了认知"
- 金句是社交货币的基本单位："收集信息不是学习，改变行为才是"
- 读者转发的是金句，而非整篇文章

**设计原则**：
1. 每篇文章至少3-5句可单独摘录的金句
2. 金句必须简洁有力（15-25字）
3. 金句必须表达反常识洞见
4. 金句必须能代表读者的认知升级

### 分享动机触发

读者分享的3种动机：

1. **身份表达**："我是实践者，不是学习者"
   - 通过转发展示自己的新身份

2. **社交价值**："这个洞见太有价值了，我要分享给朋友"
   - 通过转发提供价值，建立社交关系

3. **认知优越**："我比你先发现了这个真理"
   - 通过转发展示自己的洞察力

**设计技巧**：
- 标题必须能代表读者的认知升级
- 金句必须能被读者当作自己的观点分享
- 内容必须能让读者"装"得自然

## 高级技巧

### 反常识洞见的寻找方法

1. **逆流行**：主流观点的对立面
   - "停止学习新技能" vs "终身学习"
   - "努力工作是个陷阱" vs "勤奋是美德"

2. **寻本质**：现象背后的底层逻辑
   - "学习不是收集信息而是改变行为"
   - "行动力不是动力而是选择的能力"

3. **跨学科**：其他领域的原理迁移
   - 从认知科学到写作
   - 从编程到行动力设计

### 模块化架构原则

每个模块控制在4-5个信息点，符合7±2工作记忆限制：

```
## [模块标题：点出核心洞见]
> "引用金句" —— 出处

[先说错误理解]
但真相是：[给出正确理解]
举个例子：[具体案例]

*金句提炼*
```

**结构优势**：
- 可插拔：每个模块独立，可灵活组合
- 可复制：相同的结构确保输出质量稳定
- 可记忆：标准化结构降低认知负荷

### 视觉层次系统

引导注意力的排版设计，降低认知负荷：

| 视觉元素 | 作用 | 心理学原理 |
|----------|------|------------|
| 120px 空心标题字 | 第一眼吸引注意 | 大尺寸+高对比度 = 视觉捕捉 |
| 绿色背景的关键词 | 突出核心概念 | 颜色对比 = 信息分层 |
| 倾斜5°的短语 | 打破视觉单调 | 非对称设计 = 认知唤醒 |
| 40px 行高的正文 | 降低阅读疲劳 | 黄金行距（1.6-1.8）= 舒适阅读 |
| 段落间距60px | 创造认知呼吸空间 | 留白 = 降低认知负荷 |

### 语气控制的3层结构

1. **第一层：朋友式**
   - 用口语化表达："你可能觉得..."
   - 用反问句："为什么？"
   - 用承认错误："我曾犯过同样的错误"

2. **第二层：叛逆者**
   - 挑战主流观点："学习是个骗局"
   - 打破认知惯性："你错了"
   - 揭露真相："真正厉害的人从不上课"

3. **第三层：权威感**
   - 用数据支撑："研究表明..."
   - 用权威引用："认知科学告诉我们..."
   - 用逻辑推导："这意味着..."

**关键原则**：
- 语气必须在3层之间自然切换
- 朋友式建立信任，叛逆者打破认知，权威感传递知识
