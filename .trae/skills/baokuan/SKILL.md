## 📖 认知工程学核心原则

### 不是写作技巧，而是认知重构

我们的内容生产系统基于认知工程学，核心逻辑是：

> **外在结果只是内在认知的投射。**

因此，我们的内容不教你"做什么"，而是重构你"是谁"——身份优先于行动。所有行为改变的前提，是认知框架的改变。

### 6大认知法则（爆款文案生成）

1. **反常识法则**：打破认知惯性，制造认知冲突
2. **身份锁定法则**：识别痛点+反直觉建议
3. **时间反差法则**：短期投入→长期回报
4. **问题揭露法则**：用问题引发自省
5. **反向认同法则**：预设消极行为，激发反击欲
6. **价值锚定法则**：降低认知负荷，提供具体价值

## 💡 使用技巧

### 1. 组合使用技能

场景2（公众号爆款文章）就是典型的多技能组合：
```
awkn-content-decomposition → awkn-viral-article → awkn-cover-image
→ awkn-article-illustrator → awkn-compress-image → awkn-post-to-wechat
```

### 2. 风格混搭

- awkn-xhs-images 支持"风格 × 布局"组合
- awkn-comic 支持"风格 × 布局 × 尺寸"组合

### 3. 批量处理

- awkn-compress-image 支持目录批量压缩
- awkn-article-illustrator 自动识别需要插图的位置

### 4. 格式转换

所有输出都支持多种格式：
- 封面图：PNG / WebP / JPEG
- PPT：PPTX / PDF
- 压缩图：WebP / PNG / JPEG
