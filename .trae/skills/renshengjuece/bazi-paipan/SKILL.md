# 八字排盘脚本

## 技能定位

八字排盘是**基础工具脚本**，作为人生决策命盘系统的底层组件，不直接面向用户。

**主要功能**：
- 四柱排盘（年、月、日、时）
- 真太阳时修正（经度校正）
- 十神分析（同我、我生、我克、生我、克我）
- 五行统计（金木水火土的数量和旺衰）
- 流年流月推算

**调用方式**：由 `ren-sheng-jue-ce-ming-pan` 技能自动调用

---

## 脚本参数

```bash
python bazi_paipan.py \
  --name="姓名" \
  --gender=1 \
  --year=1990 \
  --month=3 \
  --day=15 \
  --hour=8 \
  --longitude=121.47 \
  --is-solar
```

**参数说明**：
- `--name`: 姓名
- `--gender`: 性别（1=男，0=女）
- `--year/month/day/hour`: 出生时间（年/月/日/时）
- `--longitude`: 出生城市经度
- `--is-solar`: 启用真太阳时修正

---

## 经度参考

详细的城市经度数据见 `assets/longitude.txt`，包含：
- 4个直辖市
- 27个省会城市及主要城市
- 60+个其他主要城市
- 真太阳时修正公式和示例

**真太阳时修正公式**：
```
偏移分钟 = (城市经度 - 120.0) × 4
```

**示例**：
- 北京：偏移 = (116.40 - 120.0) × 4 = -14.4分钟
- 上海：偏移 = (121.47 - 120.0) × 4 = +5.88分钟
- 成都：偏移 = (104.07 - 120.0) × 4 = -63.72分钟

## 注意事项

1. **时辰准确性**：时辰必须精确到分钟，影响排盘精度
2. **真太阳时修正**：不同经度地区需要真太阳时修正，否则时辰不准确
3. **公历/农历明确**：必须明确说明是公历还是农历
4. **经度数据**：使用 `assets/longitude.txt` 中的准确经度数据
