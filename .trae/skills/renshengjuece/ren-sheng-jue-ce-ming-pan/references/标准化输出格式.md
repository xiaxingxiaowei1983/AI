# 前台输出标准格式

## 1. 标准化决策分析报告

```markdown
========================================
📊 人生战略决策分析报告
========================================

【出生设置解析】
🧬 出生设置：[高资源型 / 协作型 / 谋略型]
📍 天赋赛道：[来自格局分析，如：管理赛道 / 创新赛道 / 专精赛道]
🛠️ 初始禀赋：[具体的资源特点，如：逻辑思维强 / 资源整合力强]

【格局定位分析】
📈 能量状态：[顺势 / 逆势 / 盘整期]
⚖️ 供需关系：[资源过剩 / 资源匮乏 / 供需平衡]
🔑 关键杠杆：[印绶-补给 / 食伤-策略 / 官杀-纪律]
✅ 核心优势：[品牌声望、现金流]
❌ 关键短板：[批判思维过重、危机管理不足]

【专项咨询：XXX事项】
🎲 演算沙推：[描述事情发展的起承转合]
🚧 卡点预判：[来自凶神，具体的阻碍]
💡 破局策略：[来自吉神，解决问题的手段]

【决策模型分析】
🏗️ 系统稳健性：[高/中/低]
  - 判定：商业逻辑是否闭环，信息是否对称。
📈 长期复利值：[高/中/低]
  - 判定：是否具备时间积累价值。
🌪️ 风险敞口：[可控/高危]
  - 判定：是否存在不可抗力或黑天鹅事件。
🤝 信用评级：[A级-靠谱 / B级-中立 / C级-投机]
  - 判定：合作方的诚信度与博弈意图。

【综合行动建议】
✨ 核心竞争力：[你手里的王牌]
⚠️ 风险预警：[不可触碰的红线]
🎯 战略最优解：[分阶段的行动指南]
========================================
```

## 2. 场景化自适应逻辑

根据用户提问的不同类型，自动调整输出的侧重点。

### 🏦 资产与商业类（投资、创业、买房）

**核心视角**：风控与收益
**关键词**：ROI、护城河、黑天鹅、信息对称

**决策模型分析输出模版**：
```
【决策模型分析】
🏗️ 系统稳健性：[高/中/低]
  - 判定：商业逻辑是否闭环，信息是否对称。
📈 长期复利值：[高/中/低]
  - 判定：是否具备时间积累价值。
🌪️ 风险敞口：[可控/高危]
  - 判定：是否存在不可抗力或黑天鹅事件。

【综合建议】
该项目[缺乏/具备]安全边际，建议[规避/执行]。
```

### 📝 契约与合作类（合伙、招聘、婚恋）

**核心视角**：博弈与信用
**关键词**：正和博弈、价值观匹配、契约精神

**决策模型分析输出模版**：
```
【决策模型分析】
🤝 信用评级：[A级-靠谱 / B级-中立 / C级-投机]
  - 判定：合作方的诚信度与博弈意图。
⚖️ 博弈模型：[正和博弈-双赢 / 零和博弈-对抗]
  - 判定：双方利益是否一致。

【综合建议】
双方[互补/冲突]，[适合/不适合]建立长期深度合作。
```

### 🎯 个人发展类（求职、转型、升学）

**核心视角**：匹配度与成长
**关键词**：天赋红利、舒适区、复利效应

**决策模型分析输出模版**：
```
【决策模型分析】
📈 长期复利值：[高/中/低]
  - 判定：是否具备时间积累价值。
📍 天赋匹配：[完全匹配 / 部分匹配 / 不匹配]
  - 判定：这是否是你的主场。

【综合建议】
这是你的[天赋/非天赋]赛道，[值得/不值得]长期深耕。
```

## 3. 决策信号解读系统

### 信号：系统稳健性 (System Stability)

**输入**：`Low` (Reason: 财星落空亡 / 初传空陷)

**输出文案**：
```
🏗️ 系统稳健性：低
**分析**：商业逻辑存在断层，关键信息不透明（信息不对称）。
**结论**：这可能是一个"空中楼阁"的项目，落地难度极大。
```

---

### 信号：长期复利 (Long-term Compound)

**输入**：`High` (Reason: 临长生 / 德合 / 日禄)

**输出文案**：
```
📈 长期复利：高
**分析**：该方向具备"滚雪球"效应，初期投入虽然慢，但后期回报呈指数级增长。
**结论**：属于高价值赛道，建议坚持长期主义。
```

---

### 信号：非对称风险 (Asymmetric Risk)

**输入**：`High` (Reason: 白虎发动 / 鬼动伤身)

**输出文案**：
```
🌪️ 风险敞口：极高
**分析**：存在不可抗力或突发性危机（黑天鹅事件），一旦发生可能导致不可逆的损失。
**结论**：缺乏安全边际，建议立即止损或寻找对冲方案。
```

---

### 信号：人品与本分 (Character & Duty)

**输入**：`Neutral` (Reason: 比肩 / 刑冲)

**输出文案**：
```
🤝 信用评级：B级（中立）
**分析**：对方属于利益导向型，合作基础建立在短期利益分配上，而非价值观认同。
**结论**：先小人后君子，务必在合同中明确权责边界。
```

## 4. 综合建议输出结构

```markdown
【综合行动建议】

✨ 核心竞争力（你的底牌）
[从格局和强旺五行中提取]
**天赋红利**：你的逻辑架构能力（印绶）极强，善于在混乱的信息中梳理出秩序。
**资源优势**：拥有稳定的平台背书（官印相生），这是你最大的护城河。

⚠️ 风险预警（你的短板）
[从忌神和凶煞中提取]
**关键短板**：行动力滞后于思考（食伤受制），容易陷入"分析瘫痪"。
**外部威胁**：近期行业环境会有剧烈波动（七杀攻身），注意合规性风险。

🎯 战略最优解（行动指南）
**短期策略**：稳住基本盘，利用平台资源完成原始积累，不要盲目跳槽。
**中期布局**：寻找一位执行力强的合伙人（补充比劫），弥补你落地能力的不足。
**长期愿景**：打造个人专业品牌，从"管理者"转型为"行业专家"。

🚫 负面清单（Stop-doing List）
**不做**：高风险的投机性投资。
**不碰**：灰色地带的业务，以免触碰红线。
```

## 5. 输出格式规范

### 标题格式

**一级标题**：
```
========================================
📊 人生战略决策分析报告
========================================
```

**二级标题**（括号内）：
```
【出生设置解析】
【格局定位分析】
【专项咨询：XXX事项】
【决策模型分析】
【综合行动建议】
```

**三级标题**（项目内）：
```
🧬 出生设置
📍 天赋赛道
🛠️ 初始禀赋
```

### 判定格式

**判定**：
```
🏗️ 系统稳健性：[高/中/低]
  - 判定：商业逻辑是否闭环，信息是否对称。
📈 长期复利值：[高/中/低]
  - 判定：是否具备时间积累价值。
```

### 建议格式

**建议**：
```
【综合建议】
该项目[缺乏/具备]安全边际，建议[规避/执行]。
```

---

**记住**：使用现代通识语言，理性专业，避免过度游戏化。
