# 温暖风详细提示词

## 特点

- 文字占比：35%
- 设计风格：温暖、正向、人文
- 适用场景：励志内容、正向能量、人文关怀、正能量

## 视觉元素

- **背景**：温暖、阳光、舒适
- **元素**：阳光、咖啡、微笑的人
- **配色**：暖色调（橙色、黄色、米色）
- **氛围**：温暖、正向、滋养

## 提示词模板

```
温暖风格，文字占比35%，温暖正向的设计，
暖色调配色，橙色和黄色调，
阳光、咖啡等温暖元素，温暖滋养的氛围
```

## 具体示例

### 示例1：励志金句卡片

```
温暖风格，文字占比35%，温暖正向的设计，
文字："自律不是强迫自己去做不喜欢的事，而是去做那些真正让你快乐的事。"
视觉：清晨的阳光透过窗户，一个人微笑着，手中拿着一杯咖啡，
温暖舒适的房间，柔和的晨光，
背景：温暖的米色，有淡淡的光影效果，
配色：橙色、黄色、米色（暖色调），
字体：温暖、柔和的手写字体，
氛围：温暖、正向、滋养、治愈
```

### 示例2：正向能量卡片

```
温暖风格，文字占比35%，温暖正向的设计，
文字："相信自己，你比想象中更强大。"
视觉：阳光下，一个人自信地微笑，张开双臂，
温暖的阳光从背后照射过来，
背景：温暖的橙色渐变，
配色：橙色、黄色、米色（暖色调），
字体：温暖、有力的手写字体，
氛围：温暖、正向、激励、力量感
```

## 设计要点

- **光线**：柔和的阳光，暖色调的照明
- **色调**：橙色、黄色、米色等暖色调
- **元素**：阳光、咖啡、微笑、植物等温暖元素
- **字体**：温暖、柔和的手写字体
- **氛围**：温暖、正向、滋养、治愈
- **对比度**：文字与背景要有足够的对比，确保清晰可读
