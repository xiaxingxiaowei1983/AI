# 教科书风格详细提示词

## 特点

- 文字占比：30%
- 设计风格：类似课本、教学资料，清晰、严谨、有逻辑感
- 适用场景：知识科普、教程、学术内容、方法论分享

## 视觉元素

- **背景**：干净的纸张纹理或简洁色块
- **排版**：清晰的标题、分层结构、编号列表
- **配色**：蓝色、灰色、米色等教科书常用色
- **图标**：简单的线条图标、箭头、标注

## 提示词模板

```
教科书风格，文字占比30%，类似高中物理课本的排版，
清晰的标题结构，分层内容，蓝色和米色配色，
背景有淡淡的纸张纹理，
简洁的线条图标和箭头标注，
有逻辑感的层次结构，
专业的学术氛围
```

## 具体示例

### 示例1：知识科普卡片

```
教科书风格，文字占比30%，类似高中生物课本的排版，
清晰的标题"细胞分裂"，
分层内容：间期、前期、中期、后期、末期，
蓝色和米色配色，
背景有淡淡的纸张纹理，
简洁的细胞分裂示意图，
简单的线条箭头标注各个阶段，
专业的学术氛围
```

### 示例2：方法论卡片

```
教科书风格，文字占比30%，类似学习方法论书籍的排版，
清晰的标题"5步学习法"，
分层内容：Step1-Step2-Step3-Step4-Step5，
灰色和米色配色，
背景有淡淡的纸张纹理，
简洁的流程图示意图，

```

## 设计要点

- **标题**：使用较大字号，加粗，颜色稍深
- **正文**：使用中等字号，黑色或深灰色
- **图标**：简洁的线条图标，不要过于花哨
- **背景**：淡淡的纸张纹理，不要过于复杂
- **配色**：蓝色、灰色、米色等教科书常用色，避免过于鲜艳
- **布局**：清晰的层次结构，使用编号列表或项目符号
