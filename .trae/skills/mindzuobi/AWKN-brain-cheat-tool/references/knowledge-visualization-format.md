## 基本结构
知识点可视化图片需包含以下元素：
- **知识节点**：核心概念/知识点（使用图标+文字表示）
- **关系连线**：展示知识间的逻辑关系（层级、因果、关联等）
- **层次结构**：从核心到细节的层次展开
- **交互提示**：视觉上暗示可交互的区域（如箭头、加号、手指图标）
- **说明文字**：简洁的文字说明（每个节点5-15字）
- **配色区分**：不同类型/层次的知识使用不同颜色

## 可视化类型

### 1. 思维导图式
```
            [核心主题]
               |
      +--------+--------+
      |        |        |
   [分支1]   [分支2]   [分支3]
      |
   [子节点]
```
适用场景：展示知识的层级结构和分类

### 2. 流程图式
```
[步骤1] → [步骤2] → [步骤3] → [步骤4]
              ↓
           [分支路径]
```
适用场景：展示知识的逻辑流程和步骤

### 3. 网络关系图
```
        [概念A]
       /  |  \
   [B]  [C]  [D]
    |    |
   [E]  [F]
```
适用场景：展示知识间的多维度关联

### 4. 时间轴式
```
[阶段1] → [阶段2] → [阶段3] → [阶段4]
  ↓         ↓         ↓         ↓
[知识点]  [知识点]  [知识点]  [知识点]
```
适用场景：展示知识的学习路径或发展历程

## 交互设计规范

### 交互类型
- **点击展开**：节点可点击展开更多子节点（用"+"图标或箭头提示）
- **悬停显示**：鼠标悬停时显示详细说明（用问号或信息图标提示）
- **拖拽重组**：节点可拖拽调整位置（用抓取手势图标提示）
- **缩放导航**：支持整体缩放查看细节（用放大镜图标提示）

### 交互提示的视觉设计
- **图标位置**：在节点的右上角或右下角
- **图标样式**：简洁的线性图标，颜色与节点区分
- **图标大小**：节点尺寸的20-30%
- **状态反馈**：可交互节点使用略微加粗的边框或阴影

### 交互反馈设计
- **悬停状态**：节点颜色加深或添加光晕效果
- **点击状态**：节点添加选中框或高亮边框
- **展开状态**：动态显示展开动画（箭头旋转、连线延伸）

## 视觉设计要求

### 节点设计
- **形状**：
  - 核心节点：圆形或圆角矩形（尺寸最大）
  - 主要节点：圆角矩形（尺寸中等）
  - 次要节点：小圆形或胶囊形（尺寸最小）
- **尺寸比例**：核心:主要:次要 = 3:2:1
- **边框**：主要节点2-3px边框，次要节点1px边框
- **阴影**：核心节点添加柔和阴影，增强层次感

### 连线设计
- **线条粗细**：根据关系强度调整（1-3px）
- **线条样式**：
  - 直接关系：实线
  - 间接关系：虚线
  - 因果关系：箭头线
  - 循环关系：双向箭头
- **线条颜色**：使用中性色（灰色）或节点配色之一
- **线条弯曲**：使用平滑曲线，避免尖锐折角

### 布局原则
- **视觉平衡**：整体构图对称或中心辐射
- **留白充足**：节点间距离至少为节点尺寸的1.5倍
- **对齐规范**：相同层级的节点对齐，增强秩序感
- **阅读顺序**：符合从左到右、从上到下的阅读习惯

### 文字规范
- **字体**：无衬线字体（如Arial、Helvetica、PingFang SC）
- **字号**：
  - 核心节点：18-24pt
  - 主要节点：14-16pt
  - 次要节点：10-12pt
- **颜色**：白色或深灰色，确保可读性
- **对齐**：居中对齐
- **字数限制**：每个节点不超过15字

## 配色方案

### 按知识层级配色
- **核心层**：主色调（深色系，如深蓝 #1A237E）
- **主要层**：辅助色（中色系，如橙色 #FF6F00）
- **次要层**：点缀色（浅色系，如青色 #00BFA5）

### 按知识类型配色
- **概念类**：蓝色系（理性、抽象）
- **方法类**：绿色系（行动、实践）
- **案例类**：橙色系（具体、生动）
- **工具类**：紫色系（技能、工具）
- **总结类**：红色系（重点、警示）

### 配色示例
```python
# 方案1：专业商务风
核心节点：深蓝 (#1565C0)
主要节点：浅蓝 (#42A5F5)
次要节点：灰蓝 (#90CAF9)
连线：灰色 (#B0BEC5)

# 方案2：清新教育风
核心节点：深绿 (#2E7D32)
主要节点：浅绿 (#66BB6A)
次要节点：嫩绿 (#A5D6A7)
连线：浅灰 (#E0E0E0)

# 方案3：现代科技风
核心节点：深紫 (#6A1B9A)
主要节点：中紫 (#AB47BC)
次要节点：浅紫 (#CE93D8)
连线：灰紫 (#E1BEE7)
```

## 示例模板

### 模板1：核心知识点思维导图
```
画面描述：
- 中心：大型圆形节点，显示书籍核心主题，深蓝色背景，白色文字，添加"+"交互提示图标
- 第一层：5个圆角矩形节点，围绕中心排列，每个对应一个核心观点，橙色背景，白色文字
- 第二层：每个第一层节点下方连接2-3个小圆形节点，表示支撑论点，青色背景，深色文字
- 连线：从中心向外辐射的平滑曲线，使用灰色
- 交互提示：每个节点右上角添加"+"图标，提示可展开更多内容
- 背景：纯白色或浅灰色渐变，突出节点
- 整体布局：中心辐射式，对称平衡
- 配色：核心层深蓝，主要层橙色，次要层青色
```

### 模板2：知识关系网络图
```
画面描述：
- 布局：散点式分布，多个节点通过连线形成网络
- 核心节点：位于画面中央，尺寸最大，深紫色背景，显示核心概念
- 主要节点：分布在核心节点周围，中等尺寸，紫色背景，显示相关概念
- 次要节点：散布在外围，小尺寸，浅紫色背景，显示细节知识点
- 连线：节点间使用不同样式的连线（实线、虚线、箭头线）表示不同关系
- 交互提示：部分节点添加"?"图标，提示悬停显示详细说明
- 整体氛围：现代科技感，层次丰富
- 配色：深紫→中紫→浅紫的渐变层次
```

### 模板3：知识流程图
```
画面描述：
- 布局：从左到右的流程式布局
- 节点：圆角矩形，从左到右依次排列，每个节点代表一个学习阶段
- 连线：使用带箭头的实线连接节点，表示流程顺序
- 分支：在某个节点后添加分支路径，用虚线箭头连接
- 交互提示：每个节点右侧添加"→"图标，提示可点击进入下一步
- 时间轴：在节点下方添加简短的时间或步骤标记
- 整体节奏：从简单到复杂，节点尺寸逐渐增大或颜色逐渐加深
- 配色：从浅绿到深绿的渐变，表示知识深度的增加
```

### 模板4：交互式知识卡片
```
画面描述：
- 布局：卡片式网格布局，每个知识点以卡片形式展示
- 卡片：圆角矩形，白色背景，带轻微阴影
- 卡片内容：
  * 上部：图标/插图（占卡片高度的40%）
  * 中部：标题文字（10-12pt）
  * 下部：简短说明（8-10pt）
- 交互提示：卡片右下角添加"点击查看详情"按钮
- 悬停效果：卡片边框高亮，轻微放大
- 卡片排列：2行3列或3行2列的网格
- 整体风格：简洁现代，信息清晰
- 配色：每个卡片使用不同类型的配色（概念-蓝、方法-绿、案例-橙）
```

## 制作注意事项
1. **交互性优先**：确保交互提示清晰可见，让用户知道哪里可以点击或悬停
2. **层次分明**：通过尺寸、颜色、位置等方式清晰展示知识层级
3. **简洁有力**：每个节点的文字简练，避免冗长
4. **色彩统一**：整套图片使用统一的配色方案，增强系列感
5. **可扩展性**：预留空间，便于后续添加新的知识点
6. **响应式设计**：考虑在不同尺寸屏幕上的显示效果
