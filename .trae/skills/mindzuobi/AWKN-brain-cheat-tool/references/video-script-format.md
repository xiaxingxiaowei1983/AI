## 脚本结构
视频脚本包含以下部分：
- 视频信息：总时长、风格定位、目标受众
- 分镜头表：具体画面与文案的逐帧呈现
- 制作备注：特别说明或注意事项

## 分镜头要素
每个分镜头需包含以下字段：
| 字段 | 说明 | 示例 |
|------|------|------|
| 镜头序号 | 连续编号 | 01 |
| 时长 | 该镜头持续秒数 | 3s |
| 画面内容 | 详细的视觉描述 | 翻书特写，阳光洒在书页上 |
| 旁白文案 | 该段落的配音文字 | 读书，是与伟大的灵魂对话 |
| 运镜方式 | 推、拉、摇、移等 | 缓慢推近 |
| 画面风格 | 实拍/动画/图文混排 | 实拍+动画叠加 |
| 配乐建议 | 情绪、节奏、乐器 | 温暖的钢琴背景乐 |

## 时长分配
### 60秒短视频
- 开头（前5-8秒）：吸引注意，提出问题
- 发展（中间40-45秒）：展开论述，呈现观点
- 结尾（最后7-10秒）：总结升华，行动号召

### 90秒短视频
- 开头（前8-10秒）：悬念导入
- 发展（中间65-70秒）：3-4个核心观点依次展开
- 结尾（最后10-12秒）：金句收尾，引导关注

### 120秒视频
- 开头（前10-15秒）：场景代入
- 发展（中间85-95秒）：深度展开，案例支撑
- 结尾（最后15-20秒）：完整总结，多重引导

## 画面描述规范
描述需包含以下维度：
- **主体内容**：人物、物体、场景（如"一位女性坐在落地窗前阅读"）
- **景别角度**：远、中、近、特写，仰拍、平视、俯拍（如"中景，平视"）
- **运动方式**：推、拉、摇、移、跟、升降（如"缓慢推向书架"）
- **色彩调性**：冷暖、明暗、饱和度（如"暖色调，明亮柔和"）
- **氛围营造**：通过光影、构图传达的情绪（如"温馨、充满希望"）
- **文字叠加**：是否需要字幕、标题（如"叠加文字：突破边界"）

### 描述示例
```
镜头01：
画面内容：特写镜头，一本书缓缓打开，阳光从窗外洒在书页上，尘埃在光束中飞舞。书页上的文字逐渐清晰。
景别角度：特写，略微俯拍
运动方式：缓慢推近
色彩调性：暖黄色调，明亮柔和
氛围营造：宁静、充满期待
文字叠加：无
```

## 旁白文案要求
- **字数控制**：每秒2-3字（30秒镜头不超过90字）
- **语言风格**：简洁有力，富有节奏感
- **逻辑连贯**：各镜头间形成清晰的故事线
- **情感共鸣**：使用第二人称或讲故事的方式，增强代入感
- **金句植入**：在关键节点插入金句，便于传播

### 文案节奏
```
开头：快速切入，制造悬念（语速稍快）
发展：层层递进，展开论述（语速适中，重点强调）
结尾：金句收尾，情感升华（语速放缓，留有余韵）
```

## 配乐建议
### 情绪分类
- **温暖治愈**：钢琴、吉他、弦乐（如《菊次郎的夏天》风格）
- **激情励志**：管弦乐、鼓点、电子（如《The Spectre》风格）
- **专业理性**：钢琴、合成器、低音（如《Interstellar》配乐）
- **轻松活泼**：尤克里里、口琴、打击乐（如《Despacito》前奏）

### 节奏搭配
- 开头：音量渐入，营造氛围
- 发展：与旁白节奏呼应，高潮处增强
- 结尾：逐渐减弱，留有余韵

## 示例模板

### 示例1：60秒短视频脚本
```
视频信息：
- 总时长：60秒
- 风格定位：温暖治愈，知识分享
- 目标受众：职场人士、终身学习者

分镜头表：

镜头01：
时长：5s
画面内容：城市夜景，一个人在办公室加班，疲惫地看向窗外。
景别角度：中景，侧拍
运动方式：固定镜头
色彩调性：冷色调，略带灰暗
氛围营造：孤独、迷茫
旁白文案：你是否也有这样的时刻？
配乐建议：缓慢的钢琴前奏，略带忧伤

镜头02：
时长：8s
画面内容：人物拿起一本书，翻开第一页。镜头特写书页上的文字。
景别角度：特写，俯拍
运动方式：缓慢推近
色彩调性：暖黄色调逐渐增强
氛围营造：期待、温暖
旁白文案：阅读，是与伟大的灵魂对话。
配乐建议：钢琴旋律开始上扬

镜头03：
时长：12s
画面内容：快速剪辑：地铁上阅读的年轻人、图书馆里的学生、家中独处的老人。
景别角度：多景别混剪
运动方式：快速切换
色彩调性：统一的暖色调
氛围营造：普遍、共鸣
旁白文案：无论身在何处，一本书就能带你穿越时空。
配乐建议：节奏加快，加入弦乐

镜头04：
时长：10s
画面内容：特写人物专注的侧脸，眼中反射着光芒。
景别角度：近景，侧拍
运动方式：缓慢推近
色彩调性：明亮暖色
氛围营造：专注、沉浸
旁白文案：在文字中，你遇见更好的自己。
配乐建议：高潮段落，管弦乐加入

镜头05：
时长：15s
画面内容：场景快速切换：城市→自然→历史→未来，展现阅读的时空穿越感。
景别角度：全景
运动方式：流畅的转场
色彩调性：渐变暖色
氛围营造：宏大、震撼
旁白文案：不设限，才能看见更大的世界。
配乐建议：音乐达到高潮

镜头06：
时长：10s
画面内容：回到现实，人物合上书，微笑着看向窗外，窗外已是阳光明媚。
景别角度：中景，正拍
运动方式：固定镜头
色彩调性：明亮的暖色
氛围营造：希望、力量
旁白文案：从今天开始，每天1小时，3个月后遇见全新的自己。
配乐建议：音乐逐渐减弱，钢琴收尾

制作备注：
- 镜头03使用快速蒙太奇手法，增强视觉冲击力
- 金句"不设限，才能看见更大的世界"需突出显示
- 整体色调从冷转暖，体现心态转变
```

```
- 目标受众：企业管理者、知识工作者

制作备注：
- 增加1-2个案例镜头，增强说服力
- 数据可视化呈现（如图表、动画）

```

## 制作注意事项
1. **节奏把控**：避免平均分配时长，重点内容给予更多时间
2. **画面连贯**：相邻镜头间要有逻辑衔接，避免突兀
3. **文字辅助**：重要观点用字幕突出，但避免文字过多
4. **情绪曲线**：整体遵循"吸引→共情→说服→行动"的情绪走向
5. **多版本准备**：根据平台特点（抖音/视频号/B站）调整节奏和风格
