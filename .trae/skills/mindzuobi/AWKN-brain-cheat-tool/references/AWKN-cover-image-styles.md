## 商务专业风格

### elegant（优雅简洁）

**特征**：
- 配色：黑白灰 + 金色/银色点缀
- 元素：几何图形、线条、留白
- 字体：无衬线字体，简洁有力
- 氛围：高端、专业、精致

**适用场景**：
- 商业文章
- 高端品牌
- 专业报告
- 领导力主题

**配色方案**：
```
主色：#1A1A1A（深黑）
辅色：#F5F5F5（浅灰）
强调色：#C9A962（金）
```

---

### bold-editorial（大胆编辑）

**特征**：
- 配色：高对比度，黑白红
- 元素：大字标题、粗线条、几何图形
- 字体：粗体无衬线，视觉冲击力强
- 氛围：大胆、有力、杂志感

**适用场景**：
- 杂志封面
- 新闻文章
- 评论文章
- 观点表达

**配色方案**：
```
主色：#000000（纯黑）
辅色：#FFFFFF（纯白）
强调色：#FF3B30（红）
```

---

### vector-illustration（矢量插画）

**特征**：
- 配色：明亮活泼，多色搭配
- 元素：矢量插画、扁平化图形
- 字体：简洁无衬线
- 氛围：现代、时尚、插画感

**适用场景**：
- 插画文章
- 设计类内容
- 创意主题
- 品牌故事

**配色方案**：
```
主色：#2C3E50（深蓝）
辅色：#E74C3C（红）、#F39C12（橙）、#27AE60（绿）
```

## 教育科普风格

### flat-doodle（扁平涂鸦）

**特征**：
- 配色：柔和明亮，低饱和度
- 元素：手绘涂鸦、扁平化图标
- 字体：圆润无衬线
- 氛围：轻松、有趣、亲切

**适用场景**：
- 教育科普
- 轻松内容
- 入门教程
- 儿童内容

**配色方案**：
```
主色：#3498DB（蓝）
辅色：#E74C3C（红）、#F1C40F（黄）、#2ECC71（绿）
```

---

### chalkboard（黑板教学）

**特征**：
- 配色：黑板绿 + 粉笔白/黄
- 元素：粉笔笔触、手写字体
- 字体：手写风格
- 氛围：教学感、经典、亲切

**适用场景**：
- 教育培训
- 学习笔记
- 教程类内容
- 知识分享

**配色方案**：
```
主色：#2D5A27（黑板绿）
辅色：#FFFFFF（粉笔白）、#F7E06D（粉笔黄）
```

---

### sketch-notes（手绘笔记）

**特征**：
- 配色：铅笔灰 + 彩色标注
- 元素：手绘线条、箭头、框图
- 字体：手写字体
- 氛围：笔记感、真实、亲切

**适用场景**：
- 学习笔记
- 思维导图
- 知识总结
- 个人成长

**配色方案**：
```
主色：#555555（铅笔灰）
辅色：#E74C3C（红）、#3498DB（蓝）、#F39C12（黄）
```

---

### notion（Notion 风格）

**特征**：
- 配色：简洁灰白 + 偶尔彩色
- 元素：图标、emoji、简洁排版
- 字体：简洁无衬线

主色：#FFFFFF（白）

## 科技技术风格

- 字体：等宽字体
- 氛围：技术、专业、科技感

- 技术文档

- 科技文章
- 系统架构

**配色方案**：
```
主色：#0066CC（蓝图蓝）
辅色：#FFFFFF（白线）、#004499（深蓝）

---

- 元素：电路图、数据流、几何图形
- 字体：科技感字体
- 氛围：AI、科技、未来感

- AI 主题
- 机器学习
- 科技文章
- 未来科技

```
主色：#00D4FF（科技蓝）
辅色：#7B2CBF（霓虹紫）、#00FF99（荧光绿）

## 风格选择指南

### 根据内容类型选择

| 内容类型 | 推荐风格 | 避免风格 |
|---------|---------|---------|
| 商业/职场 | elegant, bold-editorial, vector-illustration | playful, pixel-art |
| 教育/培训 | chalkboard, sketch-notes, flat-doodle | dark-atmospheric, retro |
| 科技/AI | blueprint, intuition-machine, minimal | watercolor, vintage |
| 创意/艺术 | fantasy-animation, playful, watercolor | blueprint, minimal |
| 生活方式 | warm, nature, retro | blueprint, dark-atmospheric |
| 数据/分析 | editorial-infographic, minimal, elegant | playful, fantasy-animation |
