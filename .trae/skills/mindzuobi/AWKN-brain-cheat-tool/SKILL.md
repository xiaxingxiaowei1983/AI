# 大脑作弊器 - 完整解决方案

## 任务目标
本 Skill 提供**一站式解决方案**，包含：
- 深度阅读并拆解书籍核心内容
- 生成公众号爆款文章（认知工程学+AWKN爆文逻辑）
- 生成可交互的知识点可视化图片
- 生成视频脚本
- **创作知识漫画**（8-16页，合并PDF）
- **生成文字卡片**（11种风格）
- **咪蒙标题大师**（生成爆款标题和图文正文）
- **文章封面生成器**（2.35:1 电影感）
- **发布到微信公众号**（文章/图文/漫画）

## 操作步骤

### 标准流程（第一步-第四步）

#### 第一步：书籍内容拆解

1. **接收用户输入**：
   - 如果是文档文件（PDF/DOCX/TXT/EPUB），调用 `scripts/document_reader.py` 提取文本内容
   - 如果是纯文本内容，直接使用

2. **深度拆解内容**：
   - 书籍主题定位（1句话）
   - 核心观点（3-5个，每个观点附带1-2个完整故事案例）
   - 内容结构（章节概要 + 关键段落索引）
   - 精选金句（10-20条，保留完整表述 + 上下文）
   - 关键案例节选（完整故事化内容）
   - 数据和引用（原文中的关键数据、权威引用）

3. **风格推荐**：
   - 推荐2-4种可视化风格（参考 AWKN-brain-cheat-tool/references/visualization-styles.md）
   - 每种风格简要说明推荐理由和适合的场景

4. **输出并等待用户确认**：
   ```
   【书籍拆解报告】
   主题定位、核心观点、内容结构、精选金句、关键案例、数据和引用

   【风格推荐】
   1. 极简Ins风 - 推荐理由：xxx，适合场景：xxx
   2. 知识图谱风 - 推荐理由：xxx，适合场景：xxx
   3. 杂志排版风 - 推荐理由：xxx，适合场景：xxx

   ✅ 第一步：书籍内容拆解 已完成

   是否进入第二步：生成公众号爆款文章？
   （回复"是"或"第二步"继续，"否"或"跳过"结束）
   ```

**⚠️ 重要：必须等待用户确认后再执行第二步**

#### 第二步：生成公众号爆款文章

1. **内容分析与提炼**：
   - 确定核心主题、核心洞见、目标受众、情绪痛点（4个核心要素）
   - 访问原文提取丰富素材（故事案例、金句、数据、场景）

2. **标题生成**：
   - 使用7大公式生成标题（参考 AWKN-brain-cheat-tool/references/viral-logic.md）
   - 提供3-5个标题选项供用户选择

3. **正文结构（6模块说服术）**：
   - 参考 AWKN-brain-cheat-tool/references/viral-logic.md 中的完整格式规范
   - **重要**：钩子开场必须从"钩子开场示例库"中随机选择，或根据书籍内容生成全新的钩子开场，禁止重复使用同一个钩子开场
   - 钩子开场（前150字）：打破认知惯性（反向认同/自我揭露/反常识宣言/直击痛点/数据震撼型/场景代入型/金句开场型）
   - 痛点共鸣（150-200字）：建立情感连接
   - 价值承诺（100字）：降低阅读门槛
   - 核心内容（800-1000字）：重构认知框架
   - 金句提炼（3-5句用*斜体*标记）：制造传播点
   - 赋权结尾（150字）：激发行动/思考

4. **转发文案生成**（约140字内，不被微信折叠）
5. **摘要生成**（100字内）

6. **可选：咪蒙标题优化**：
   - 基于咪蒙5大核心法则生成10个标题（参考 AWKN-brain-cheat-tool/references/miming-methodology.md）
   - 每个标题标注使用的法则和子策略
   - 提供标题吸引力分析和心理触发点说明

7. **输出**：
   ```
   【标题选项】
   选项1：xxx
   选项2：xxx
   选项3：xxx

   【正文】
   [1500字标准版文章]

   【转发文案】（约140字内）
   xxx

   【摘要】（100字内）
   xxx

   【咪蒙标题优化】（可选）
   10个爆款标题选项及分析

   ✅ 第二步：生成公众号爆款文章 已完成

   是否进入第三步：生成知识可视化图片？
   （回复"是"或"第三步"继续，"否"或"跳过"结束）
   ```

#### 第三步：生成知识可视化图片

1. **阶段一：风格预览生成**
   - 为推荐的2-4种风格，每种风格生成1张预览图片
   - 调用 `scripts/image_generator.py`
   - 展示预览图片，询问用户确认风格

2. **阶段二：系列图片生成**（必须执行5次）：
   - 生成5张系列图片（思维导图、关系网络图、流程图、交互卡片、时间轴）
   - 调用 `scripts/image_generator.py` 5次
   - 保存为 image1.png ~ image5.png

3. **阶段三：公众号插图生成**（必须执行3次）：
   - 生成3张公众号正文插图（适合手机竖屏阅读）
   - 调用 `scripts/image_generator.py` 3次
   - 风格参考 AWKN-brain-cheat-tool/references/visualization-styles.md（28种风格）
   - 保存为 AWKN-illust-01.png、AWKN-illust-02.png、AWKN-illust-03.png

4. **封面生成**（自动执行）：
   - 调用智能体生成文章封面（2.35:1，电影感）
   - 支持多种风格（参考 AWKN-brain-cheat-tool/references/AWKN-cover-image-styles.md）
   - 保存为 cover.png

5. **生成图文正文**：
   - 提炼核心观点（3-5个）
   - 使用短句（每句不超过20字）
   - 金句点缀（3-5句）
   - 数据/案例支撑（简要）
   - 行动呼吁（结尾）
   - 限制200字内

   File: [AWKN-illust-03.png] - 正文插图3

   【文章封面】
   File: [AWKN-cover.png] - 文章封面（2.35:1 电影感）

   【图文正文】
   标题、正文（200字内）

   ✅ 第三步：生成知识可视化图片 已完成

#### 第四步：生成视频脚本

1. **设计视频结构**：
   - 总时长：90秒
   - 开头分配：5-10秒（钩子开场）
   - 发展分配：70-80秒（核心内容展示）
   - 结尾分配：5-10秒（行动呼吁/总结）

2. **生成分镜头脚本表**（参考 AWKN-brain-cheat-tool/references/video-script-format.md）：
   - 镜头序号、画面内容描述、旁白文案、时长估算、配乐建议

3. **输出并等待用户确认**：
   ```
   【视频脚本】
   镜头序号 | 画面内容描述 | 旁白文案 | 时长估算 | 配乐建议
   1        | [开场画面]  | [开场旁白] | 5秒    | 轻快
   ...
   总时长：90秒

   ✅ 第四步：生成视频脚本 已完成

   是否进入扩展功能？
   （回复"是"或"扩展"继续第五步，"否"或"跳过"结束）

### 扩展功能（第五步-第七步）

#### 第五步：创作知识漫画

1. **内容分析**：
   - 确定漫画主题和页数（建议8-16页）
   - 确定叙事方案（按时间线/按主题/按角色）

2. **风格与布局选择**：
   - 风格：8种（参考 AWKN-brain-cheat-tool/references/style-ohmsha.md）
   - 布局：6种（标准/电影/密集/大图/混合/长条）
   - 尺寸：3种（3:4/4:3/16:9）

3. **生成分镜方案**：
   - 生成详细的面板描述、文字内容、视觉提示词（参考 AWKN-brain-cheat-tool/references/analysis-framework.md）
   - 参考分镜模板（AWKN-brain-cheat-tool/references/storyboard-template.md）

4. **生成漫画页面**：
   - 为每个页面生成详细的视觉提示词（参考 AWKN-brain-cheat-tool/references/base-prompt.md）
   - 生成图像（使用智能体原生图像生成能力）

5. **合并为PDF**：
   - 调用 `scripts/merge-to-pdf.ts` 合并所有漫画页面
   - 生成 `{topic-slug}.pdf`

6. **输出并等待用户确认**：
   ```
   【知识漫画】

3. **布局设计**：

   - 尺寸：1080x1080（方形）、1080x1350（竖版）、1200x675（横版）
   - 文字占比：30-40%
