## 项目创建计划

### 1. 创建项目目录结构
- 创建 `ai-proxy` 目录
- 在 `ai-proxy` 目录下创建 `public` 子目录

### 2. 创建核心文件
- **server.js**：后端代理服务，使用 Node.js + Express
- **.env.example**：环境变量模板文件
- **.gitignore**：Git 忽略文件配置
- **package.json**：项目依赖和脚本配置
- **public/chat.html**：前端测试页面

### 3. 实现 server.js
- 安装必要的依赖：express, axios, cors, dotenv
- 配置环境变量加载
- 实现 CORS 配置
- 实现 POST /api/ai 路由
- 实现 DashScope REST API 调用
- 实现错误处理和响应

### 4. 创建前端测试页面
- 实现极简界面：输入框 + 发送按钮 + 结果区域
- 实现 API 调用逻辑
- 添加安全注释说明

### 5. 验证配置
- 检查所有文件是否符合要求
- 确保安全规范得到遵循
- 提供启动和使用说明

- 清晰的注释说明
