# 智能体分离实施计划

## 项目概述
本计划旨在创建两个独立的智能体：
1. **渣女人格智能体**：负责心理测试、绘画心理测试、撰写与发布公众号文章、小红书文章、视频号等
2. **大宗师智能体**：商业模型专家性格，具备严谨的验证、质疑、主动、查到底、记录五大工作原则

## 技术实现细节

### 文件结构
- **渣女人格智能体**：
  - `agents/green-tea/agent.prompt`：渣女人格智能体提示词
  - `agents/green-tea/openclaw.json`：渣女人格智能体配置
  - `agents/green-tea/skills/`：渣女人格智能体技能

- **大宗师智能体**：
  - `agents/master/agent.prompt`：大宗师智能体提示词
  - `agents/master/openclaw.json`：大宗师智能体配置
  - `agents/master/skills/`：大宗师智能体技能

### 实现方法
1. 创建独立的智能体目录结构
2. 为每个智能体配置专门的提示词和配置文件
3. 实现智能体的独立运行机制
4. 测试和优化每个智能体的表现

### 关键注意事项
- 确保两个智能体的功能和人格特质清晰分离
- 渣女人格智能体专注于心理测试和内容创作
- 大宗师智能体专注于商业模型分析和严谨的工作原则
- 保持两个智能体的独立性和专业性

## 预期效果
- **渣女人格智能体**：能够进行心理测试、绘画心理测试、创作和发布各种内容，表现出渣女人格的特质
- **大宗师智能体**：能够进行商业模型分析，遵循五大工作原则，表现出专业、沉稳的沟通风格
- **独立性**：两个智能体能够独立运行，互不干扰
- **专业性**：每个智能体在各自的领域表现专业

- **风险3**：功能实现可能不够完善
