# 飞书机器人邀请外部群操作指南（小白版）

## 📋 任务概述
帮助您将飞书机器人添加到外部群（跨组织群），无需编写任何代码，使用最简单的邀请链接方式。

## 🎯 推荐方案：邀请链接（最简单，无需代码）

### 为什么选择这个方案？
- ⭐ 难度最低，只需要复制粘贴
- ⭐ 不需要任何技术知识
- ⭐ 适用于所有场景
- ⭐ 官方支持的方式

---

## 📝 详细操作步骤

### 第一步：获取机器人邀请链接

1. **打开飞书开放平台**
   - 在浏览器中访问：https://open.feishu.cn
   - 使用您的飞书账号登录

2. **进入您的应用**
   - 点击右上角"控制台"
   - 找到您创建的机器人应用（App ID: cli_a91012cd0ab89bc9）
   - 点击进入应用详情

3. **找到机器人配置**
   - 在左侧菜单中找到"机器人"选项
   - 点击进入机器人配置页面

4. **复制邀请链接**
   - 在机器人页面中找到"邀请链接"或"添加到群"部分
   - 点击"复制链接"按钮
   - 链接格式类似：`https://applink.feishu.cn/client/bot/chat?bot_id=xxx`

### 第二步：在外部群中发送邀请链接

1. **打开目标外部群**
   - 打开飞书App或网页版
   - 进入您想要添加机器人的外部群

2. **发送邀请链接**
   - 在群聊输入框中粘贴刚才复制的链接
   - 可以添加说明文字，例如：
     ```
     点击下面的链接添加机器人：
     https://applink.feishu.cn/client/bot/chat?bot_id=xxx
     ```

3. **群成员点击链接**
   - 群成员点击链接后，会自动添加机器人到群聊

### 第三步：验证机器人是否添加成功

1. **检查群成员列表**
   - 在群信息中查看成员列表
   - 确认机器人已出现在成员列表中

2. **测试机器人功能**
   - 在群中@机器人，发送测试消息
   - 确认机器人能够正常回复

## ⚠️ 常见问题解决

### 问题1：找不到邀请链接
**解决方案**：
- 确保您的应用已启用"机器人"功能
- 在飞书开放平台 → 应用详情 → 机器人 → 开启机器人能力
- 如果没有看到邀请链接，可能需要先发布应用

### 问题2：链接点击后没反应
**解决方案**：
- 确认链接是否完整复制
- 尝试在飞书App中打开链接（而不是浏览器）
- 确认机器人应用已通过审核并发布

### 问题3：外部群无法添加机器人
**解决方案**：
- 外部群需要群管理员同意添加机器人
- 可以联系群管理员协助操作
- 或者使用"群二维码"方式邀请

---

## 🔄 备选方案（如果邀请链接不成功）

### 方案二：群二维码方式
1. 在飞书开放平台获取机器人的群二维码
2. 将二维码发送到外部群
3. 群成员扫码添加机器人

### 方案三：管理员邀请
1. 联系外部群的管理员
2. 请管理员在群设置中手动添加机器人
3. 管理员路径：群设置 → 添加机器人 → 选择您的机器人
