# OpenClaw 问题修复 - 实施计划

## 问题复盘

### 核心问题
1. **语言显示问题**：界面不显示中文，每次重启都变为英文
2. **设置持久化问题**：语言设置不持久化，需要手动切换才能生效
3. **CSP错误**：Google字体无法加载，违反Content Security Policy
4. **服务器启动问题**：OpenClaw网关服务启动后立即退出

### 根本原因分析
1. **语言设置不同步**：存在两个localStorage存储位置（`clawpal_language`和`openclaw.control.settings.v1`）
2. **CSP配置不完整**：缺少对Google字体域名的允许
3. **服务器配置问题**：OpenClaw网关服务启动配置有问题
