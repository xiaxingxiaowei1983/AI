# 会员权益和积分系统实现计划

## 1. 积分系统架构
- 创建积分状态管理（localStorage + React state）
- 积分规则：
  - 注册会员：赠送 1000 积分
  - 每日打卡：赠送 100 积分
  - 充值：1元 = 500 积分
  - 使用数字分身：扣除 100 积分

## 2. 修改 LoginModal.tsx
- 注册成功后自动赠送 1000 积分
- 登录成功后检查积分状态
- 显示当前积分余额

## 3. 创建积分管理组件 (PointsManager.tsx)
- 显示积分余额
- 每日打卡按钮（每日限1次）
- 充值功能（模拟支付）
- 积分使用记录

## 4. 修改 AvatarChat.tsx
- 发送消息前检查积分余额
- 积分不足时显示提示
- 扣除 100 积分后调用 API
- 显示剩余积分

## 5. 创建打卡弹窗 (CheckInModal.tsx)
- 每日签到功能
- 连续签到奖励（可选）
- 签到记录

## 6. 创建充值弹窗 (RechargeModal.tsx)
- 充值金额选择（1元、5元、10元等）
- 模拟支付流程
- 充值成功后更新积分

## 7. 更新导航栏
- 显示积分余额
- 快捷入口到积分管理

## 技术实现
- 使用 localStorage 持久化积分数据
- 添加积分变动历史记录
- 防止重复打卡（日期标记）
- 积分不足时的友好提示
