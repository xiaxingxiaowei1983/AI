# 晚上进化专家（Night Evolution Expert）

## 核心功能

### 1. 进化模式管理
- **模式切换**：根据时间自动切换进化模式（标准/连续）
- **时段识别**：准确识别晚上时段（19:00-次日7:00）
- **模式配置**：提供灵活的进化模式配置选项

### 2. 连续进化优化
- **进化序列优化**：优化连续进化的序列，确保进化的连贯性和效率
- **资源管理**：智能管理系统资源，避免过度消耗
- **疲劳检测**：检测进化疲劳，适时调整进化强度

### 3. 晚上进化策略
- **快速见效策略**：优先进化能够快速见效的能力
- **深度进化策略**：在合适的时机进行深度进化，优化核心能力
- **平衡进化策略**：平衡快速见效和深度进化，确保整体进化效果

### 4. 进化质量控制
- **ADL合规性检查**：确保晚上进化符合反进化锁定协议
- **价值评估强化**：强化对晚上进化候选的价值评估
- **稳定性保障**：保障晚上进化的稳定性，避免过度进化导致的问题

## 技术实现

### 1. 时间管理
- **时段识别**：使用系统时间识别当前时段
- **进化节奏控制**：根据时段控制进化的节奏和强度
- **任务调度**：智能调度进化任务，优化进化效率

### 2. 策略实现
- **策略引擎**：实现多种进化策略，根据情况自动选择
- **决策算法**：使用智能算法决定进化的方向和强度
- **适应性调整**：根据进化效果自动调整策略参数

### 3. 质量保障
- **ADL验证**：集成ADL验证机制，确保进化的合规性
- **价值评估**：使用强化的价值评估机制，确保进化的价值
- **稳定性监控**：监控进化对系统稳定性的影响，及时调整进化策略

## 使用方法

### 触发方式
- **自动触发**：根据时间自动激活晚上进化模式
- **手动触发**：通过特定指令手动激活晚上进化模式
- **条件触发**：在特定条件下自动激活晚上进化模式

### 期望输入
- **进化目标**：指定晚上进化的具体目标
- **时间限制**：指定晚上进化的时间限制
- **资源限制**：指定晚上进化的资源限制

### 输出形式
- **进化计划**：生成详细的晚上进化计划
- **进化状态**：实时反馈晚上进化的状态和进度
- **进化报告**：生成晚上进化的详细报告

## 约束条件

1. **ADL合规性**：所有晚上进化必须符合反进化锁定协议
2. **稳定性优先**：晚上进化必须以系统稳定性为优先考虑
3. **效率优化**：晚上进化必须优化进化效率，提高进化效果
4. **资源合理性**：晚上进化必须合理使用系统资源，避免过度消耗
