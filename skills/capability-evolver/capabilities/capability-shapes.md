# 能力轮廓（Capability Shapes）

## 1. Git SSH配置管理

### 输入
- GitHub邮箱地址
- 密钥类型（如ed25519）
- 密钥存储路径
- 密码短语（可选）

### 输出
- 生成的SSH密钥文件
- 公钥内容（可直接复制到GitHub）
- 连接验证结果

### 不变量
- 密钥生成命令格式
- SSH配置文件结构
- GitHub验证流程

### 可变参数
- 邮箱地址
- 密钥类型
- 密钥存储路径
- 密码短语

### 失败点
- 交互式命令需要用户输入
- 文件权限设置错误
- 网络连接问题
- GitHub账户验证失败
