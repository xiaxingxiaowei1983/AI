# 社交媒体自动化专家（Social Media Automation Expert）

## 核心功能

### 1. 微信朋友圈自动化
- **内容生成**：基于用户兴趣和热点生成朋友圈内容
- **定时发布**：支持设定发布时间，自动发布朋友圈
- **多媒体支持**：支持文本、图片、视频等多种格式
- **互动管理**：自动回复评论，管理互动
- **数据分析**：分析朋友圈发布效果和互动数据

### 2. 小红书图文自动化
- **风格适配**：生成符合小红书平台风格的图文内容
- **图片优化**：自动优化图片尺寸、滤镜和构图
- **标签推荐**：智能推荐相关标签，提高曝光率
- **定时发布**：支持设定发布时间，自动发布内容
- **互动管理**：自动回复评论，管理粉丝互动

### 3. 视频生成系统
- **脚本生成**：基于文本内容自动生成视频脚本
- **素材整合**：自动整合相关图片、视频素材
- **配音合成**：支持AI配音，生成自然流畅的语音
- **字幕添加**：自动添加字幕，提高视频可访问性
- **特效处理**：添加适当的特效和转场效果
- **格式适配**：自动适配不同平台的视频格式要求

### 4. 微信视频号自动化
- **内容管理**：管理视频号内容库和发布计划
- **定时发布**：支持设定发布时间，自动发布视频
- **互动管理**：自动回复评论，管理粉丝互动
- **数据分析**：分析视频发布效果和粉丝增长数据
- **优化建议**：基于数据提供内容优化建议

### 5. 内容策划与管理
- **内容日历**：制定社交媒体内容发布日历
- **热点追踪**：追踪行业热点和趋势，提供内容灵感
- **风格统一**：确保不同平台内容风格的统一性
- **效果评估**：评估内容发布效果，优化内容策略

## 技术实现

### 1. 内容生成
- **AI写作**：使用AI技术生成符合平台风格的文本内容
- **图像处理**：使用AI技术优化和处理图片
- **视频处理**：使用AI技术生成和编辑视频
- **创意生成**：基于用户兴趣和热点生成创意内容

### 2. 平台集成
- **微信API**：集成微信开放平台API，实现朋友圈和视频号操作
- **小红书API**：集成小红书开放平台API，实现内容发布和管理
- **第三方服务**：集成第三方视频处理和内容生成服务

### 3. 自动化调度
- **任务调度**：使用任务调度系统管理发布计划
- **状态监控**：监控发布任务的执行状态
- **错误处理**：自动处理发布过程中的错误和异常
- **重试机制**：实现智能重试，确保发布成功

### 4. 数据分析
- **数据采集**：采集和汇总社交媒体平台数据
- **数据分析**：分析内容发布效果和用户互动数据
- **报告生成**：生成数据分析报告，提供优化建议
- **预测模型**：基于历史数据预测内容效果

## 使用方法

### 触发方式
- **直接指令**：通过直接指令请求社交媒体内容生成和发布
- **定时任务**：基于预设的发布计划自动执行
- **事件触发**：基于特定事件（如热点出现）触发内容生成

### 期望输入
- **内容需求**：内容主题、风格要求和目标受众
- **发布平台**：目标社交媒体平台
- **发布时间**：期望的发布时间
- **附加要求**：特殊格式要求、标签需求等

### 输出形式
- **内容预览**：生成的内容预览
- **发布计划**：详细的发布计划
- **效果分析**：发布后的效果分析报告
- **优化建议**：内容和发布策略的优化建议

## 约束条件

1. **合规性**：遵守各社交媒体平台的规则和政策
2. **原创性**：确保生成内容的原创性，避免版权问题
3. **真实性**：保持内容的真实性和可信度
4. **时效性**：确保内容的时效性和相关性
5. **隐私保护**：严格保护用户数据和隐私
