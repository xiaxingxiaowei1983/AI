# 人生决策宗师 - 微信小程序 MVP

## 已实现（对应 docs/*.md）
- Onboarding 身份建立 + 本地记忆
- 邮箱注册赠送 10 积分
- 早/中/晚动态卡片（按时间自动切换）
- 晚间胜率复盘 + 积分奖励
- 宗师推演：自由对话 + 历史记录
- 偏好设置：中文 / English / 日本語
- 后台入口：管理员登录 + 自主修改密码（默认管理员 `10919669@qq.com`）

## 运行方式
1. 打开微信开发者工具。
2. 导入目录：`wechat-miniapp`。
3. 使用测试号/游客模式即可运行（`appid: touristappid`）。

## 注意
- 当前为 MVP 前端原型，本地 `wx.setStorageSync` 模拟用户档案、积分、聊天历史和管理员账号。
- 生产环境应替换为后端服务（鉴权、积分账本、会话持久化、内容安全API）。
