## 问题分析

用户要求修改性别选择的UI设计，具体要求：
1. 将下拉菜单改为小的男/女点选按钮
2. 将性别选择放在基本信息的右边
3. 用色块衬底

## 解决方案

### 修改 OnboardingPage.tsx 文件

#### 1. 调整布局结构
- **修改用户信息输入区域**：将年龄和性别放在同一行
- **布局结构**：
  ```tsx
  <div className="grid grid-cols-2 gap-4">
    <div>
      {/* 年龄输入 */}
    </div>
    <div>
      {/* 性别选择按钮 */}
    </div>
  </div>
  ```

#### 2. 实现性别点选按钮
- **替换 select 元素**：用两个按钮代替下拉菜单
- **添加样式**：为按钮添加色块衬底
- **状态管理**：更新 handleGenderChange 函数

#### 3. 更新样式
- **按钮样式**：添加色块背景、圆角等样式
- **响应式设计**：确保在不同屏幕尺寸下显示正常
- **交互效果**：添加选中状态的视觉反馈

#### 4. 修改事件处理
- **更新 handleGenderChange 函数**：处理按钮点击事件
- **保持状态管理**：确保用户信息能够正确存储

### 具体修改内容

1. **修改布局结构**：使用 grid 布局将年龄和性别放在同一行
2. **实现性别按钮**：创建两个按钮，分别对应男/女
3. **添加样式**：为按钮添加色块衬底和交互效果
4. **更新事件处理**：修改 handleGenderChange 函数
5. **保持其他功能**：确保用户信息的存储和传递功能正常

### 预期效果

- 年龄输入框在左，性别选择按钮在右
- 性别选择使用小的男/女点选按钮
- 按钮有色块衬底，选中时有视觉反馈
- 整体布局美观，符合用户要求

### 技术实现

- 使用 Tailwind CSS 实现布局和样式
- 使用 React 状态管理处理用户输入
- 保持代码的可读性和可维护性
