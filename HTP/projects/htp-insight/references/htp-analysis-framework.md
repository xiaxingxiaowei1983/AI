# 核心维度

## 整体布局维度

### 三元素的整体构图
**评估指标**：
- 平衡构图：三元素协调分布，画面平衡
- 不平衡构图：某些元素过度突出或被边缘化
- 拥挤构图：三元素之间距离过近，缺乏空间
- 疏远构图：三元素之间距离过远，缺乏连接
- 倾斜构图：整体画面或某些元素倾斜

**解读方法**：
- 评估三元素之间的整体关系和构图平衡
- 分析拥挤或疏远反映的心理空间分布
- 注意倾斜或偏斜反映的压力或不稳定感
- 结合三元素的位置关系判断整合程度

**适用场景**：
- 所有个体
- 初步评估整体心理状态

### 三元素的空间分布
**评估指标**：
- 位置关系：上下左右布局、相对距离、覆盖遮挡
- 距离远近：三元素之间的物理距离
- 连接方式：是否有道路、线条等元素连接
- 空间占比：每个元素占据的空间比例

**解读方法**：
- 分析三元素的位置关系反映的价值排序
- 评估距离远近反映的亲密程度或隔离感
- 判断三元素的整合程度或冲突程度
- 结合空间占比判断各元素的重要性

**适用场景**：
- 所有个体
- 评估家庭关系、自我认知、人际关系

# 房子特征维度

### 房子的位置
**评估指标**：
- 居中位置：家庭中心感强、平衡需求、安全感
- 偏左位置：与过去的关系、传统导向、情感取向
- 偏右位置：与未来的关系、创新导向、理性取向
- 偏上位置：幻想倾向、理想化、逃避现实
- 偏下位置：现实导向、安全感需求、根基重视

**解读方法**：
- 分析房子位置反映的家庭价值取向
- 注意极端位置的特殊含义
- 结合其他元素的位置判断整体平衡

**参考文档**：
[htp-symbolism-dictionary.md](htp-symbolism-dictionary.md) 中的房子位置部分

**适用场景**：
- 所有个体
- 家庭关系评估、安全感评估

### 房子的大小
**评估指标**：
- 大房子：家庭重要性高、家庭期望大、安全感需求
- 小房子：家庭功能弱、家庭忽视、独立性
- 适中房子：家庭平衡、现实感知、健康功能

**解读方法**：
- 根据纸张大小判断房子相对大小
- 结合房子结构、装饰等细节综合评估
- 注意大小与其他元素的交互作用

**参考文档**：
[htp-symbolism-dictionary.md](htp-symbolism-dictionary.md) 中的房子大小部分

**适用场景**：
- 所有个体
- 家庭功能评估、自我认知评估

### 房子的结构完整性
**评估指标**：
- 完整结构：屋顶、门、窗户、墙壁、地基都存在
- 缺失元素：缺少屋顶、门、窗户等关键元素
- 破损结构：屋顶塌陷、墙壁破损、门窗破损
- 复杂结构：多个房间、多层结构、装饰丰富

**解读方法**：
- 分析结构完整性反映的家庭功能状态
- 注意缺失元素可能象征的家庭问题
- 评估破损程度反映的家庭创伤或压力
- 判断复杂程度反映的家庭复杂性和适应能力

[htp-symbolism-dictionary.md](htp-symbolism-dictionary.md) 中的房子元素部分（屋顶、门、窗户、墙壁、烟囱、地基等）

**适用场景**：
- 所有个体
- 家庭功能评估、安全感评估、创伤识别

### 房子的细节特征
**评估指标**：
- 装饰程度：简单朴素到复杂装饰
- 对称性：对称设计到不对称设计
- 细节密度：细节稀疏到细节丰富
- 风格特征：现代、传统、抽象、具体

**解读方法**：

**适用场景**：
- 所有个体
- 情感表达评估、审美取向评估、文化背景识别

## 情绪表达维度

### 整体情绪基调
**评估指标**：
- 积极基调：温暖、明亮、充满希望
- 消极基调：阴暗、沉重、充满压抑
- 中性基调：平和、客观、无明显情绪
- 混合基调：情绪矛盾、不协调、复杂

**解读方法**：
- 分析整体色调、线条力度、构图等综合判断情绪基调
- 评估情绪基调的强度和稳定性
- 注意矛盾或混合情绪反映的内在冲突

**适用场景**：
- 所有个体
- 情绪状态评估、心理状态初步评估

### 能量水平
**评估指标**：
- 高能量：线条有力、构图向上、充满活力
- 中等能量：线条适中、构图平衡、稳定状态
- 低能量：线条柔弱、构图垂落、缺乏活力

**解读方法**：
- 分析线条力度和方向判断能量水平
- 评估构图特征反映的活力和积极性
- 注意能量水平与情绪基调的一致性

**适用场景**：
- 所有个体
- 能量水平评估、活力程度评估、抑郁识别

### 焦虑水平
**评估指标**：
- 高焦虑：大量涂抹、断续线条、拥挤构图、尖刺特征
- 中等焦虑：局部涂抹、适度断续、轻微拥挤
- 低焦虑：无涂抹、连续线条、宽松构图、圆润特征
- 无焦虑：绘画流畅、稳定、平衡

**解读方法**：
- 分析涂抹程度和范围判断焦虑水平
- 评估线条连续性和尖锐度反映的不稳定性和防御性
- 判断构图特征反映的紧张感和冲突程度
- 注意焦虑水平的分布（整体或局部）

**适用场景**：
- 所有个体

### 防御机制

**解读方法**：
- 分析房子和人物特征反映的防御水平
- 评估技术特征（线条、涂抹）反映的不安全感和保护需求
- 判断三元素之间的距离反映的隔离程度
- 注意防御机制的合理性和适应性

**适用场景**：
- 所有个体
- 防御机制评估、安全感评估、保护需求识别

## 发展维度

### 心理发展阶段
**评估指标**：

### 成长导向

## 综合应用

### 完整评估流程

1. **整体观察**：初步评估整体构图、三元素关系、情绪基调
2. **逐元素分析**：分别分析房子、树木、人物的特征和含义
3. **技术特征评估**：分析线条、涂抹、细节、空白等技术特征
4. **情绪和发展评估**：评估情绪表达、发展水平、适应能力
5. **关系和比例分析**：分析三元素之间的关系、大小、位置
6. **背景适配**：考虑年龄和文化背景对解读的影响
7. **综合整合**：整合各维度信息，形成完整的心理画像
8. **风险评估**：识别潜在的风险指标和需要关注的问题

### 示例：成人平衡状态

**整体布局**：居中位置，适中大小，平衡构图

**房子特征**：
- 位置：偏左，结构完整，有门有窗
- 大小：适中，装饰适度
- 解读：家庭关系稳定，安全感良好，家庭功能健康

**树木特征**：
- 位置：居中，树干强壮，树冠茂盛
- 大小：适中，向上生长
- 解读：自我认知清晰，生命力旺盛，成长导向积极

**人物特征**：
- 位置：偏右，身体完整，直立姿态
- 大小：适中，面部表情平和
- 解读：身份认同清晰，心理状态稳定，态度积极

**技术特征**：
- 线条：强有力，连续流畅
- 涂抹：无或极少
- 细节：适度，均匀分布
- 空白：适度利用，不拥挤不疏远

**情绪和发展**：
- 情绪基调：积极，温暖
- 能量水平：中高
- 焦虑水平：低
- 防御机制：适中
- 发展水平：符合年龄阶段
- 自我发展：自我接纳
- 适应能力：良好
- 成长导向：强

**整体画像**：心理状态平衡，家庭关系稳定，自我认知清晰，成长导向积极，适应能力良好，无明显风险指标。

### 示例：青少年焦虑状态

**整体布局**：偏上位置，过小元素，倾斜构图

**房子特征**：
- 位置：偏上，无门或小门，窗户缺失或很小
- 大小：过小
- 解读：家庭功能弱，安全感不足，可能与家庭隔离

**树木特征**：
- 位置：偏左，树干细弱，树冠稀疏
- 大小：过小
- 解读：自我认知不清晰，生命力不足，可能存在自卑或退缩

**人物特征**：
- 位置：偏右，身体完整但小，姿态略显僵硬
- 大小：过小
- 解读：身份认同可能不清晰，可能存在焦虑或紧张

**技术特征**：
- 线条：柔弱，局部断续
- 涂抹：局部中度涂抹，集中在人物和树木
- 细节：不均匀，某些细节过度，某些缺失
- 空白：大量空白，元素集中在小区域

**情绪和发展**：
