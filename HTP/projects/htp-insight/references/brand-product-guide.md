# 产品介绍

## 项目简介

**你的画，照见你的灵魂**，是一个融合了心理学理论与 AI 技术的智能分析工具。通过分析用户手绘的绘画作品，系统能够深入解读个体的心理状态、人格特质、内在矛盾与成长潜力，为用户和心理咨询师提供双重视角的专业洞察。

这不仅仅是一个心理测试工具，更是一面照见内心的镜子——**让每一笔绘画，都成为理解心灵的窗口。**

---

## 核心亮点

### 智能视觉识别
- **深度特征提取**：自动识别整体布局、线条与笔触、细节特征、整体印象
- **技术特征分析**：捕捉线条力度、连续性、粗细、方向、笔触压力等绘画技术特征
- **整体布局解析**：分析画面位置、大小比例、空间关系的空间语义

### 专业心理学解读
- **象征体系映射**：基于心理学理论，将视觉特征映射到心理学含义
- **多维度分析框架**：
  - 整体布局维度（空间定位、大小比例、平衡感）
  - 元素特征维度（自我认知、思维模式、现实感）
  - 技术特征维度（能量水平、焦虑程度、注意力）
  - 大小与比例维度（整体大小、元素关系）
  - 位置与布局维度（画面位置、元素距离、整体构图）
  - 线条与笔触维度（线条力度、连续性、方向、笔触压力）
  - 情绪表达维度（情绪基调、防御机制）
  - 发展维度（心理阶段、自我发展、适应能力）

### 双份报告系统
- **客户洞察报告**（面向用户）：
  - 温暖友好的语言风格
  - 聚焦核心洞察和行动建议
  - 激发自我探索和成长意愿
  - HTML 格式 + 智能配图 + 长图文（2个成果）

- **专业分析报告**（面向咨询师）：
  - 详细系统的理论依据
  - 完整的评估维度和指标
  - 风险评估和专业建议
  - Markdown 格式（仅文档）

### 智能配图系统
- **统一风格选择**：所有插图使用 warm 风格（温暖、友好、人性化）
- **主题智能生成**：
  - 封面插图：整体心理状态的视觉隐喻
  - 深度洞察插图：关键意象的可视化表达
  - 行动建议插图：积极行动的视觉引导
  - 最后的邀请插图：温暖支持的视觉表达
- **warm 风格特征**：
  - 配色：暖橙色 (#ED8936)、奶油色背景 (#FFFAF0)、金黄色 (#F6AD55)、深棕色 (#744210)
  - 视觉元素：柔和的形状、友好的角色、温暖的光线、柔和的渐变
  - 适合所有年龄段：儿童、青少年、成年、老年

### HTML 响应式报告
- **温暖友好的视觉设计**：渐变背景、圆润卡片、柔和配色
- **响应式布局**：完美适配移动端和桌面端
- **长图文格式**：竖版设计，适合社交媒体分享
- **打印友好**：优化的打印样式，便于保存和分享
- **无障碍访问**：支持屏幕阅读器，符合无障碍标准

### 风险警示系统
- **自动风险识别**：检测多个高风险和中风险指标
- **紧急建议机制**：为高风险用户提供专业转介建议
- **危机干预支持**：提供心理危机热线和紧急联系方式
- **始终提醒**：本分析不能替代专业诊断
