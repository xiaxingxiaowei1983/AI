# HTP-Insight 技能逻辑与实现说明

## 一、技能概述

**HTP-Insight**（你的画，照见你的灵魂）是一个基于绘画心理学的智能分析系统。通过分析用户手绘的绘画作品（房-树-人），深度解读个体的心理状态、人格特质与成长潜力，生成双份报告（专业分析 + 客户洞察），并提供带智能配图的 HTML 输出及长图文分享格式。

### 核心价值
- **看见**：看见用户的内心世界，看见那些未被察觉的细节
- **理解**：理解用户的情绪、渴望、矛盾和挣扎
- **成长**：给用户行动的指引，陪伴用户走向更深的自我认知

## 二、技能逻辑架构

### 2.1 整体工作流

```
用户上传绘画
    ↓
图像解析与特征提取（智能体图像识别）
    ↓
基于心理学体系进行特征解读（参考象征体系词典）
    ↓
整体关系分析与综合整合
    ↓
多维度人格评估（参考分析框架）
    ↓
风险评估（参考风险警示系统）
    ↓
生成双份报告
    ├─ 专业分析报告（Markdown，仅文档）
    └─ 客户洞察报告（HTML + 智能配图 + 长图文，2个成果）
```

### 2.2 核心处理阶段

#### 阶段 1：图像解析与特征提取
**执行者**：智能体图像识别能力
**输入**：用户上传的绘画图片
**输出**：结构化的视觉特征描述

**识别维度**：
- **整体布局**：大小、位置、构图、平衡感、空间利用
- **线条与笔触**：线条力度、连续性、粗细、方向、笔触压力
- **细节特征**：各元素的细节丰富度、完整性、装饰
- **整体印象**：画面整体感觉、情感基调、情绪表达

#### 阶段 2：基于心理学体系进行特征解读
**执行者**：智能体自然语言理解 + 参考文档
**输入**：视觉特征描述
**输出**：心理学含义映射

**解读依据**：
- 参考 [references/htp-symbolism-dictionary.md](references/htp-symbolism-dictionary.md)
- 各元素的象征意义：家庭、成长、自我、关系等
- 线条与笔触的象征意义：能量水平、情绪状态、心理状态
- 大小与位置的象征意义：自我认知、价值排序、心理导向
- 细节与完整性的象征意义：观察力、表达方式、内在状态

#### 阶段 3：整体关系分析与综合整合
**执行者**：智能体逻辑推理
**输入**：各元素的心理含义
**输出**：整体心理画像

**分析维度**：
- **位置关系**：左右上下布局、距离远近、覆盖遮挡
- **大小比例**：各元素的相对大小、比例协调性
- **整体构图**：平衡感、对称性、协调性
- **价值排序**：哪些元素最突出？反映个人最重视的领域
- **整合程度**：各元素是否和谐共存？还是相互冲突？
- **心理空间分布**：画面布局反映内在心理空间的组织方式
- **动态平衡**：各元素之间的能量流动和平衡状态
- **一致性与矛盾性**：各特征的意义是否一致？是否存在冲突？

#### 阶段 4：多维度人格评估
**执行者**：智能体综合分析 + 参考文档
**输入**：整体心理画像
**输出**：系统性的多维度评估报告

**评估框架**：参考 [references/htp-analysis-framework.md](references/htp-analysis-framework.md)

**评估维度**：
1. **整体布局维度**：画面中的位置、大小比例、空间关系
2. **元素特征维度**：各元素的特征和心理含义
3. **技术特征维度**：线条质量、涂抹、细节程度、情绪表达
4. **大小与比例维度**：各元素相对于画面的整体大小、元素之间的比例关系
5. **位置与布局维度**：各元素在画面中的位置、元素之间的距离和关系、整体构图
6. **线条与笔触维度**：线条的粗细、强弱、断续；笔触的压力；线条的方向
7. **情绪表达维度**：情绪基调、防御机制
8. **发展维度**：心理阶段、自我发展、适应能力

**年龄与文化适配**：
- 考虑年龄因素：儿童/青少年/成年/老年
- 考虑文化因素：集体主义/个人主义/跨文化

#### 阶段 5：风险评估
**执行者**：智能体规则匹配 + 参考文档
**输入**：多维度评估结果
**输出**：风险等级和警示建议

**风险警示系统**：参考 [references/htp-warning-signs.md](references/htp-warning-signs.md)

**风险等级**：
- **无风险**：正常心理状态
- **低风险**：轻微情绪困扰
- **中风险**：需要关注的心理问题
- **高风险**：需要专业干预的严重心理问题

**风险指标识别**：
- 高风险指标：极端暴力、自伤或自杀意象、严重混乱和精神病性内容、极度扭曲的现实感
- 中风险指标：明显的焦虑、抑郁倾向、情绪压抑、社交退缩等

**风险处理**：
- 在报告中明确标注风险等级
- 提供专业转介建议
- 提供危机热线和紧急联系方式

#### 阶段 6：双份报告生成
**执行者**：智能体内容生成 + 图像生成
**输入**：多维度评估 + 风险评估 + 用户特征
**输出**：专业分析报告 + 客户洞察报告

**报告类型**：

**专业分析报告**（面向专业人士）：
- **格式**：Markdown 文档（仅文本，无 HTML 和配图）
- **内容**：详细系统的理论依据、完整的评估维度和指标、风险评估和专业建议
- **用途**：为心理咨询师提供专业参考

**客户洞察报告**（面向普通用户）：
- **格式**：HTML + 智能配图 + 长图文（2个成果）
  - `client-insight.html`：HTML 格式，包含智能配图
  - `client-insight-image.png`：长图文格式，适合分享
- **内容**：温暖友好的语言风格、聚焦核心洞察和行动建议、激发自我探索和成长意愿
- **配图**：4 张智能生成的插图（封面、洞察、建议、邀请）
- **风格**：根据用户年龄和文化背景智能选择

## 三、智能配图系统

### 3.1 配图位置与主题

客户洞察报告包含 4 张智能配图：

1. **封面插图**（报告开头）
   - 主题：整体心理状态的视觉隐喻
   - 示例：平静的湖面、生长的树苗、温暖的阳光、内在的花园
   - 目的：营造温暖、安全、接纳的氛围

2. **深度洞察插图**（深度洞察部分）
   - 主题：关键意象的可视化表达
   - 示例：左右分岔的道路、内在房间的隐喻、树的年轮、桥梁连接
   - 目的：帮助客户理解关键洞察

3. **行动建议插图**（行动建议部分）
   - 主题：积极行动的视觉引导
   - 示例：温暖的手、开放的门、攀登的台阶、播种的动作
   - 目的：激发行动意愿和希望

4. **最后的邀请插图**（最后的邀请部分）
   - 主题：温暖支持的视觉表达
   - 示例：阳光下的树、拥抱的场景、盛开的花朵、温柔的光芒
   - 目的：传递支持和鼓励

### 3.2 配图风格选择

根据用户特征自动选择配图风格：

**儿童客户**：
- 风格：playful 或 warm
- 特征：友好、童趣、色彩鲜艳
- 主色调：鲜亮的蓝色 (#4299E1)
- 背景色：淡黄色 (#FEFCBF)

**青少年客户**：
- 风格：flat-doodle 或 notion
- 特征：现代、简洁、清新
- 主色调：柔和的蓝绿色
- 背景色：浅灰色或米白色

**成年客户**：
- 风格：warm 或 elegant
- 特征：温暖、专业、成熟
- 主色调：暖橙色 (#ED8936)
- 背景色：米白色 (#FFFAF0) → 桃色渐变 (#FED7AA)

**老年客户**：
- 风格：watercolor 或 vintage
- 特征：柔和、怀旧、优雅
- 主色调：柔和的绿色 (#68D391)
- 背景色：淡粉色 (#FED7E2)

### 3.3 配图生成流程

```
Step 1: 分析用户特征
  - 识别用户年龄（儿童/青少年/成年/老年）
  - 识别文化背景（集体主义/个人主义/跨文化）
  - 识别心理状态（焦虑/抑郁/成长/平衡）

Step 2: 选择配图风格
  - 根据用户特征自动选择合适的风格

Step 3: 生成配图提示词
  - 基于报告内容、用户特征、风格选择生成详细的图像生成提示
  - 包含：主题、风格特征、情感基调、颜色方案、构图要求、文化适配

Step 4: 生成插图
  - 使用智能体的图像生成能力，根据提示词生成插图

Step 5: 整合到 HTML
  - 将生成的插图插入到 HTML 报告的相应位置

Step 6: 生成长图文
  - 将 HTML 报告转换为长图文格式（client-insight-image.png），适合分享
```

## 五、核心代码与实现

### 5.1 实现方式选型

**重要决策**：本 Skill 不使用任何脚本，所有功能通过智能体自然语言指导完成。

**决策依据**：
1. **充分利用智能体已有能力**：
   - 图像识别能力：自动提取绘画的视觉特征
   - 自然语言理解能力：深度理解心理学理论和分析框架
   - 图像生成能力：基于报告内容智能生成配图
   - 内容生成能力：生成专业分析和客户洞察报告

2. **避免过度工程化**：
   - 不需要技术脚本处理图像识别、自然语言处理、图像生成等任务
   - 智能体已经具备这些能力，直接调用即可

3. **保持灵活性**：
   - 自然语言指导更灵活，可以根据具体情况进行调整
   - 避免硬编码的业务逻辑

    # 2. 深度洞察插图
    illustrations["insight"] = generate_image(
        theme="关键意象的可视化表达",

        emotion="理解、共鸣"

    # 3. 行动建议插图
    illustrations["action"] = generate_image(
        theme="积极行动的视觉引导",
        style=style,
        emotion="希望、激励"

    # 4. 最后的邀请插图
    illustrations["invitation"] = generate_image(
        theme="温暖支持的视觉表达",

    return illustrations

## 七、关键技术要点

### 7.1 充分利用智能体能力

**图像识别**：
- 自动提取绘画的视觉特征
- 识别整体布局、线条笔触、细节特征、整体印象

**自然语言理解**：
- 深度理解心理学理论和分析框架
- 将视觉特征映射到心理学含义
- 进行逻辑推理和综合整合

**图像生成**：
- 基于报告内容和用户特征智能生成配图
- 根据年龄和文化背景自动选择风格
- 生成 4 张主题化插图

**内容生成**：
- 生成专业分析报告（详细、系统、理论化）
- 生成客户洞察报告（温暖、友好、聚焦洞察和建议）
- 生成 HTML 格式和长图文格式

### 7.2 无脚本设计

**为什么不用脚本？**

1. **智能体已有能力**：
   - 图像识别、自然语言理解、图像生成、内容生成
   - 直接调用即可，无需重复建设

2. **避免过度工程化**：
   - 简单任务不需要复杂脚本
   - 保持技能的简洁性和可维护性

3. **保持灵活性**：
   - 自然语言指导更灵活
   - 可以根据具体情况进行调整

### 7.3 双份报告系统

**专业分析报告**：
- **目标用户**：心理咨询师、专业人士
- **内容特点**：详细、系统、理论化
- **输出格式**：Markdown 文档（仅文本）
- **核心价值**：提供专业参考依据

**客户洞察报告**：
- **目标用户**：普通用户、绘画者本人
- **内容特点**：温暖、友好、聚焦洞察和建议
- **输出格式**：HTML + 智能配图 + 长图文（2个成果）
- **核心价值**：激发自我探索和成长意愿

### 7.4 智能配图系统

**风格自适应**：
- 根据用户年龄自动选择风格
- 儿童客户：playful（友好、童趣）
- 青少年客户：flat-doodle（现代、简洁）
- 成年客户：warm/elegant（温暖、专业）
- 老年客户：watercolor（柔和、怀旧）

**主题化生成**：
- 封面插图：整体心理状态
- 深度洞察插图：关键意象
- 行动建议插图：积极行动
- 最后的邀请插图：温暖支持

### 7.5 风险警示系统

**自动识别**：
- 高风险指标：极端暴力、自伤或自杀意象、严重混乱和精神病性内容、极度扭曲的现实感
- 中风险指标：明显的焦虑、抑郁倾向、情绪压抑、社交退缩等

**风险处理**：
- 在报告中明确标注风险等级
- 提供专业转介建议
- 提供危机热线和紧急联系方式
- 始终提醒：本分析不能替代专业诊断

## 九、注意事项

1. **避免绝对化判断**：
   - 绘画分析结果仅作为探索和参考，不能作为临床诊断依据
   - 避免使用绝对化的语言（如"肯定是抑郁"、"绝对是焦虑症"）
   - 应使用探索性语言（如"可能存在抑郁倾向"、"显示一定程度的焦虑"）

2. **识别警示信号**：
   - 及时识别需要专业心理干预的警示信号
   - 在报告中明确标注，建议专业心理咨询或治疗

3. **明确边界**：
   - 始终强调分析报告的探索性和辅助性，而非诊断性
   - 提醒用户本工具不能替代专业临床诊断

4. **以绘画为核心**：
   - 所有分析围绕绘画作品展开
   - 不暴露内层技术系统（HTP、房树人分析等）
   - 对外以"绘画心理分析"为核心表达

5. **充分利用智能体能力**：
   - 准确提取绘画的视觉特征
   - 深度理解心理学理论
   - 智能生成配图和报告

6. **考虑年龄和文化背景**：
   - 在解读时始终考虑年龄和文化背景的影响
   - 调整报告风格和解读方式

7. **检测风险指标**：
   - 必须在报告中明确标注风险指标
   - 提供紧急建议和专业转介

## 十、总结

**HTP-Insight** 是一个充分利用智能体已有能力的绘画心理分析系统。通过智能体的图像识别、自然语言理解、图像生成和内容生成能力，实现了完整的绘画分析流程，包括特征提取、心理学解读、多维度评估、风险评估和双份报告生成。

**核心特点**：
1. **无脚本设计**：所有功能通过智能体自然语言指导完成
2. **双份报告**：专业分析报告（Markdown）+ 客户洞察报告（HTML + 配图 + 长图文）
3. **智能配图**：根据用户特征自动选择风格，生成 4 张主题化插图
4. **风险警示**：自动识别风险指标，提供专业转介建议
5. **温暖友好**：以"看见、理解、成长"为核心理念，无评判、有温度

**技术亮点**：
- 充分利用智能体的多模态能力
- 无需复杂脚本，保持技能简洁性
- 参考文档提供专业知识支持
- 灵活适配不同年龄段和文化背景
