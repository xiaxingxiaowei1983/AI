# HTP 智能体集成指南

## 概述

本指南详细说明了如何在 HTP 心理分析系统中集成和使用智能体提示词与工作流，确保系统能够正确分析用户的房树人绘画作品并生成结构化的分析结果。

## 核心文件

### 智能体提示词
- **文件路径**：`htp-insight-agent-prompt.md`
- **用途**：定义智能体的行为、分析方法和输出格式
- **核心功能**：指导智能体如何分析绘画作品、识别风险指标、生成分析结果

### 工作流文档
- **文件路径**：`htp-insight-workflow.md`
- **用途**：定义智能体分析的完整工作流程
- **核心功能**：详细说明从图像识别到结果输出的八个步骤

## 集成步骤

### 1. 火山方舟智能体配置

1. **登录火山方舟控制台**
   - 访问火山方舟控制台：https://console.volcengine.com/ark
   - 使用您的火山引擎账号登录

2. **创建分析智能体**
   - 点击「创建智能体」按钮
   - 填写智能体信息：
     - 名称：HTP分析智能体
     - 描述：基于房树人绘画进行心理分析
     - 提示词：复制 `htp-insight-agent-prompt.md` 文件的完整内容

3. **配置接入点**
   - 创建一个多模态接入点，支持图像输入
   - 记录接入点ID，用于后端配置

4. **获取API密钥**
   - 在火山方舟控制台的「API密钥管理」页面获取API密钥
   - 记录API密钥，用于后端配置

### 2. 后端配置

1. **创建环境变量文件**
   - 在 `backend` 目录下创建 `.env` 文件
   - 填写以下配置信息：

   ```env
   # 火山方舟核心配置
   ARK_API_KEY=your_api_key
   ARK_ANALYSIS_AGENT_ID=your_analysis_agent_id
   ARK_IMAGE_AGENT_ID=your_image_agent_id
   ARK_API_BASE_URL=https://ark.cn-beijing.volces.com/api/v3
   PORT=3000
   ```

   注意：
   - `ARK_API_KEY`：填写从火山方舟获取的API密钥
   - `ARK_ANALYSIS_AGENT_ID`：填写创建的分析智能体ID
   - `ARK_IMAGE_AGENT_ID`：填写创建的图像生成智能体ID（如果使用）

2. **启动后端服务**
   - 进入 `backend` 目录
   - 运行以下命令：

   ```bash
   node server.js
   ```

   后端服务将运行在 `http://localhost:3000`

### 3. 前端配置

1. **创建环境变量文件**
   - 在项目根目录下创建 `.env` 文件
   - 填写以下配置信息：

   ```env
   VITE_BACKEND_BASE_URL=http://localhost:3000/api/htp
   ```

2. **启动前端服务**
   - 进入项目根目录
   - 运行以下命令：

   ```bash
   npm run dev
   ```

   前端服务将运行在 `http://localhost:5173`

## 智能体提示词使用说明

### 核心功能

1. **图像识别与特征提取**
   - 识别绘画作品的整体布局、房子特征、树木特征、人物特征和技术特征
   - 提取关键视觉元素，为后续分析提供基础

2. **基于象征体系的解读**
   - 将视觉特征映射到心理学含义
   - 分析房子、树木、人物等元素的象征意义

3. **整体关系分析**
   - 分析三元素之间的互动关系
   - 综合整合所有特征，形成整体心理画像

4. **多维度人格评估**
   - 基于多个维度进行系统性评估
   - 提供全面的心理状态分析

5. **风险评估**
   - 识别潜在的心理风险指标
   - 评估风险等级，提供专业建议

6. **输出结构化结果**
   - 生成JSON格式的分析结果
   - 包含三个维度的分析内容、风险等级和插图提示词

### 输出格式

智能体将返回以下JSON格式的分析结果：

```json
{
  "section_see": "[整体画面解码，描述用户的内心世界]",
  "section_understand": "[关键意象解读，理解用户的情绪、渴望、矛盾]",
  "section_grow": "[行动建议，陪伴用户走向更深的自我认知和成长]",
  "illustrations": [
    "[用于生成疗愈插画的提示词1]",
    "[用于生成疗愈插画的提示词2]",
    "[用于生成疗愈插画的提示词3]"
  ],
  "risk_level": "low",
  "professionalReport": "[详细的专业心理分析报告内容]",
  "riskAssessment": {
    "level": "low",
    "indicators": [],
    "suggestions": ""
  }
}
```

## 工作流执行说明

### 工作流程

智能体分析遵循以下八个步骤：

1. **图像识别与特征提取**：识别绘画作品的关键视觉特征
2. **基于象征体系进行特征解读**：将视觉特征映射到心理学含义
3. **整体关系分析与综合整合**：分析三元素之间的互动关系
4. **多维度人格评估**：基于多个维度进行系统性评估
5. **风险评估**：识别风险指标，评估风险等级
6. **生成双份报告**：生成专业分析报告和客户洞察报告
7. **智能配图生成**：为前端系统提供插图提示词
8. **输出JSON格式分析结果**：整合所有结果为标准JSON格式

### 执行流程

1. **用户上传绘画作品**
   - 用户可以选择在线绘制或上传图片
   - 系统将绘画作品转换为Base64格式

2. **前端发送分析请求**
   - 调用后端的 `/api/htp/analyze` 接口
   - 发送包含图像Base64、年龄和性别的请求数据

3. **后端调用智能体**
   - 后端接收请求并验证数据
   - 调用火山方舟智能体进行分析

4. **智能体执行分析**
   - 智能体按照工作流程分析绘画作品
   - 生成结构化的分析结果

5. **后端返回结果**
   - 后端接收智能体的分析结果
   - 将结果返回给前端

6. **前端展示结果**
   - 前端根据 `risk_level` 字段决定显示哪个结果页面
   - 低/中风险：显示普通结果页
   - 高风险：显示高风险结果页
   - 用户可以选择查看专业报告

## 故障排查

### 常见问题

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| 后端服务启动失败 | 环境变量配置错误 | 检查 `.env` 文件中的配置信息是否正确 |
| 智能体分析失败 | API密钥错误或智能体配置错误 | 检查API密钥和智能体配置是否正确 |
| 前端无法获取分析结果 | 后端服务未运行或网络错误 | 确保后端服务正在运行，检查网络连接 |
| 分析结果格式错误 | 智能体提示词配置错误 | 检查 `htp-insight-agent-prompt.md` 文件中的输出格式配置 |
| 高风险结果页面未显示 | `risk_level` 字段值错误 | 确保智能体返回正确的 `risk_level` 值 |

### 调试建议

1. **查看后端日志**
   - 后端服务启动后，会在控制台输出详细的日志信息
   - 分析失败时，日志会显示具体的错误信息

2. **测试API接口**
   - 使用Postman或curl测试 `/api/htp/analyze` 接口
   - 确保接口能够正确接收请求并返回结果

3. **验证智能体配置**
   - 在火山方舟控制台测试智能体
   - 确保智能体能够正确分析绘画作品并返回结构化结果
