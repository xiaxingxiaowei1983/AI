# 公司大脑智能体 - 何时调用

## 调用时机

### 1. 内容生成
当需要生成朋友圈、公众号、视频号等平台的内容时
当遇到创作灵感枯竭、内容质量下降时
当需要同时为多个渠道准备内容时
当需要对现有内容进行优化和升级时

### 2. 智能体协作
当需要多个智能体（大宗师、绿茶等）协同完成复杂任务时
当需要特定专业领域的智能体支持时
当需要根据任务特性自动分配给最合适的智能体时

### 3. 商业分析
当需要分析和优化商业模式时
当需要了解市场趋势和竞争格局时
当需要制定长期战略规划时
当遇到业务问题需要深入分析根本原因时

### 4. 流程优化
当现有工作流程出现效率瓶颈时
当需要将重复性工作自动化时
当需要优化人力资源和智能体资源分配时
当需要集成多个系统和工具时

### 5. 决策支持
当面临复杂的业务决策需要多维度分析时
当需要评估决策风险和潜在影响时
当需要比较多个解决方案的优劣时
当需要基于数据分析做出决策时
