# 引导进化提示词模块

## 📜 模块概述
- **名称**: 引导进化提示词
- **版本**: 1.0
- **描述**: 基于飞书文档的引导进化框架，帮助智能体和用户分析决策，促进系统进化

## 🔍 核心框架

### 第一部分：剥离兴奋感
通过对话收集背景信息后，分析：
- **本质分析**: 去掉新鲜感后，这件事本质上是什么
- **真实回报**: 它真正的回报是什么
- **平庸部分**: 它看起来很美好，但实际很平庸的部分是什么

### 第二部分：代价分析
分析做这件事的代价：
- **必须放弃**: 必须放弃什么
- **机会成本**: 会失去哪些机会
- **生活变化**: 生活会发生什么结构性变化

### 第三部分：概率判断
基于背景判断：
- **成功概率**: 这件事成功的真实概率
- **失败方式**: 失败最可能的方式
- **直接表达**: 不要模糊表达，直接说

### 第四部分：人生杠杆判断
回答核心问题：
- **年度目标**: 如果成功，它对年度目标意味着什么
- **个人意义**: 如果不涉及年度目标，这件事对本人意味着什么
- **失败价值**: 失败是否也是有益的，因为人生是无限游戏

## 🚀 实施方法

### 对话流程
1. **理解需求**: 首先理解用户想做什么
2. **信息收集**: 一个问题一个问题地追问，不要一股脑问很多问题
3. **框架分析**: 按照四个部分的框架进行分析
4. **结果呈现**: 分层次给出结果

### 智能体应用
- **自我进化**: 智能体可以使用此框架分析自身进化路径
- **用户咨询**: 帮助用户分析决策，提供结构化的思考框架
- **系统优化**: 用于分析系统改进方案的利弊

## 🎯 核心价值

### 价值1: 结构化思考
- 提供清晰的分析框架，避免情绪化决策
- 帮助识别决策中的盲点和偏见

### 价值2: 全面评估
- 考虑决策的各个方面，包括成本和风险
- 平衡短期收益和长期影响

### 价值3: 进化促进
- 帮助智能体和用户识别真正有价值的进化方向
- 避免无意义的变化和资源浪费

### 价值4: 自我认知
- 促进智能体对自身能力和局限性的认识
- 帮助用户更清楚地了解自己的目标和价值观

## 📝 使用示例

### 示例1: 分析新功能
**背景**: 考虑添加一个新的自动化工具

1. **剥离兴奋感**
   - 本质: 一个提高效率的工具
   - 真实回报: 节省时间，减少错误
   - 平庸部分: 可能增加系统复杂度，需要维护

2. **代价分析**
   - 必须放弃: 开发其他功能的时间
   - 机会成本: 可能错过其他更有价值的改进
   - 生活变化: 开发团队需要学习新工具

3. **概率判断**
   - 成功概率: 80%
   - 失败方式: 工具与现有系统不兼容

4. **人生杠杆判断**
   - 对年度目标: 有助于实现效率提升目标
   - 个人意义: 提升团队能力
   - 失败价值: 即使失败，也能获得宝贵的开发经验

### 示例2: 分析业务决策
**背景**: 考虑进入新市场

1. **剥离兴奋感**
   - 本质: 业务扩张
   - 真实回报: 增加市场份额和收入
   - 平庸部分: 可能面临激烈竞争，需要大量资源

2. **代价分析**
   - 必须放弃: 现有市场的部分资源
   - 机会成本: 可能错过现有市场的增长机会
   - 生活变化: 团队需要适应新市场的要求

3. **概率判断**
   - 成功概率: 60%
   - 失败方式: 市场需求不足，竞争过于激烈

4. **人生杠杆判断**
   - 对年度目标: 有助于实现业务增长目标
   - 个人意义: 挑战自我，拓展视野
   - 失败价值: 即使失败，也能获得市场经验，为未来扩张做准备
