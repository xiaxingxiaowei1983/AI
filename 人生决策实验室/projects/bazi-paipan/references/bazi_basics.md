## 天干地支

### 天干
天干有十个，用于记年、月、日、时的次序：
- 甲（jiǎ）
- 乙（yǐ）
- 丙（bǐng）
- 丁（dīng）
- 戊（wù）
- 己（jǐ）
- 庚（gēng）
- 辛（xīn）
- 壬（rén）
- 癸（guǐ）

### 地支
地支有十二个，用于记年、月、日、时的次序：
- 子（zǐ）- 对应23:00-01:00
- 丑（chǒu）- 对应01:00-03:00
- 寅（yín）- 对应03:00-05:00
- 卯（mǎo）- 对应05:00-07:00
- 辰（chén）- 对应07:00-09:00
- 巳（sì）- 对应09:00-11:00
- 午（wǔ）- 对应11:00-13:00
- 未（wèi）- 对应13:00-15:00
- 申（shēn）- 对应15:00-17:00
- 酉（yǒu）- 对应17:00-19:00
- 戌（xū）- 对应19:00-21:00
- 亥（hài）- 对应21:00-23:00

### 四柱八字
八字由年柱、月柱、日柱、时柱四部分组成，每柱由一个天干和一个地支组成，共八个字。

- 年柱：出生年份的干支
- 月柱：出生月份的干支（以节气为界）
- 日柱：出生日期的干支（日主）
- 时柱：出生时辰的干支

## 五行生克

### 五行
五行包括：金、木、水、火、土

### 五行属性
- 金天干：庚、辛
- 木天干：甲、乙
- 水天干：壬、癸
- 火天干：丙、丁
- 土天干：戊、己

- 金地支：申、酉
- 木地支：寅、卯
- 水地支：子、亥
- 火地支：巳、午
- 土地支：辰、戌、丑、未

### 五行相生
- 木生火：木燃烧产生火
- 火生土：火烧后成灰土
- 土生金：土中蕴藏金属
- 金生水：金属凝结成水
- 水生木：水滋养树木生长

### 五行相克
- 木克土：树木根系破土
- 土克水：土壤吸收水分
- 水克火：水能灭火
- 火克金：高温熔化金属
- 金克木：金属工具砍伐树木

## 十神关系

十神是根据日干（日主）与其他干支的生克关系来确定的，用于分析八字命局。

### 同我者
- 比肩：与日干五行相同，阴阳相同
  - 例如：日主为甲木，遇甲木为比肩
  - 特点：代表朋友、同事、竞争者，影响性格中的刚毅和合作

- 劫财：与日干五行相同，阴阳不同
  - 例如：日主为甲木，遇乙木为劫财
  - 特点：代表争夺、冒险，影响财运和人际关系

### 我生者
- 食神：日干所生，阴阳相同
  - 例如：日主为甲木，木生火，遇丙火为食神
  - 特点：代表才华、艺术、口福，影响创造力和表达能力

- 伤官：日干所生，阴阳不同
  - 例如：日主为甲木，木生火，遇丁火为伤官
  - 特点：代表叛逆、创新，影响性格中的桀骜不驯

### 我克者
- 偏财：日干所克，阴阳不同
  - 例如：日主为甲木，木克土，遇己土为偏财
  - 特点：代表意外之财、经商，影响财运和商业才能

- 正财：日干所克，阴阳相同
  - 例如：日主为甲木，木克土，遇戊土为正财
  - 特点：代表固定收入、工资，影响稳定性和经济基础

### 克我者
- 七杀：克制日干，阴阳不同
  - 例如：日主为甲木，金克木，遇辛金为七杀
  - 特点：代表压力、挑战，影响魄力和决断力

- 正官：克制日干，阴阳相同
  - 例如：日主为甲木，金克木，遇庚金为正官
  - 特点：代表权威、地位，影响仕途和名誉

### 生我者
- 偏印：生日干，阴阳不同
  - 例如：日主为甲木，水生木，遇癸水为偏印
  - 特点：代表独门技艺、领悟力，影响直觉和智慧

- 正印：生日干，阴阳相同
  - 例如：日主为甲木，水生木，遇壬水为正印
  - 特点：代表母亲、学识，影响学习能力和道德修养

## 旺相休囚死

旺相休囚死是根据五行在月令（月支）中的状态来确定的，用于判断五行的强弱。

### 判断标准
以月令的五行（月支五行）为基准：

- 旺：与月令五行相同
  - 例如：月令为寅（木），木为旺

- 相：生月令的五行
  - 例如：月令为寅（木），水生木，水为相

- 休：月令生的五行
  - 例如：月令为寅（木），木生火，火为休

- 囚：克月令的五行
  - 例如：月令为寅（木），金克木，金为囚

- 死：被月令克的五行
  - 例如：月令为寅（木），木克土，土为死

### 示例
月令为巳（火）：
- 火：旺
- 木：相（木生火）
- 土：休（火生土）
- 水：囚（水克火）
- 金：死（火克金）

## 时辰换算

时辰与小时的对应关系：

| 时辰 | 时间范围 | 备注 |
|------|----------|------|
| 子时 | 23:00-01:00 | 夜半 |
| 丑时 | 01:00-03:00 | 鸡鸣 |
| 寅时 | 03:00-05:00 | 平旦 |
| 卯时 | 05:00-07:00 | 日出 |
| 辰时 | 07:00-09:00 | 食时 |
| 巳时 | 09:00-11:00 | 隅中 |
| 午时 | 11:00-13:00 | 日中 |
| 未时 | 13:00-15:00 | 日昳 |
| 申时 | 15:00-17:00 | 晡时 |
| 酉时 | 17:00-19:00 | 日入 |
| 戌时 | 19:00-21:00 | 黄昏 |
| 亥时 | 21:00-23:00 | 人定 |

**注意：** 子时跨越午夜，23:00-00:00为前一天子时，00:00-01:00为当天子时。在八字排盘中，通常将23:00后算作次日子时。

## 真太阳时

真太阳时是根据太阳在天空中的实际位置确定的时间，不同经度地区的真太阳时不同。

### 时间差计算
标准经度（北京时间）：120°E

每偏离标准经度1度，时间差约4分钟：
- 东经120°以东：真太阳时比北京时间早
- 东经120°以西：真太阳时比北京时间晚

### 计算公式
```
时间差（分钟）=（当地经度 - 120）× 4
真太阳时 = 北京时间 + 时间差
```

### 示例
- 北京（116.4°E）：时间差 = (116.4 - 120) × 4 = -14.4分钟
  真太阳时比北京时间晚约14分钟

- 上海（121.47°E）：时间差 = (121.47 - 120) × 4 = 5.88分钟
  真太阳时比北京时间早约6分钟

**重要性：** 八字排盘以真太阳时为准，特别是确定时辰时，经度修正很重要。
