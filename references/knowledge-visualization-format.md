# 知识可视化格式规范

## 思维导图格式

### 核心要求
- **中心主题**：突出显示，字体较大（16-20px）
- **分支层级**：最多3-4级，保持清晰
- **节点文字**：每个节点不超过15字
- **颜色编码**：使用2-4种主题色，区分不同类型的知识点
- **布局**：中心向外辐射，层次分明

### 示例Prompt模板
```
创建一个关于[主题]的思维导图，中心是[核心概念]，包含以下分支：
1. [第一分支主题]：
   - [子节点1]
   - [子节点2]
   - [子节点3]
2. [第二分支主题]：
   - [子节点1]
   - [子节点2]
3. [第三分支主题]：
   - [子节点1]
   - [子节点2]

风格要求：[选定风格]，使用[颜色方案]，节点大小根据重要性调整，布局整洁有序。
```

## 知识关系网络图

### 核心要求
- **节点**：表示知识点，大小反映重要性
- **连线**：表示关系，粗细反映关联强度
- **布局**：力导向布局，相关节点聚集
- **标签**：每个节点有简短标签（10字以内）
- **图例**：包含颜色和连线样式说明

### 示例Prompt模板
```
创建一个关于[主题]的知识关系网络图，包含以下知识点：
- [知识点1]
- [知识点2]
- [知识点3]
- [知识点4]
- [知识点5]

关系说明：
- [知识点1] 与 [知识点2] 是[关系类型]
- [知识点2] 与 [知识点3] 是[关系类型]
- [知识点3] 与 [知识点4] 是[关系类型]
- [知识点4] 与 [知识点5] 是[关系类型]

风格要求：[选定风格]，使用[颜色方案]，节点大小根据重要性调整，连线粗细反映关系强度。
```

## 知识流程图

### 核心要求
- **步骤**：清晰的流程步骤，使用数字编号
- **方向**：从左到右或从上到下的流向
- **形状**：不同形状表示不同类型的步骤
- **连线**：带箭头的连线，指示流程方向
- **说明**：每个步骤有简短说明（12字以内）

### 示例Prompt模板
```
创建一个关于[主题]的知识流程图，包含以下步骤：
1. [步骤1]：[简短说明]
2. [步骤2]：[简短说明]
3. [步骤3]：[简短说明]
4. [步骤4]：[简短说明]
5. [步骤5]：[简短说明]

风格要求：[选定风格]，使用[颜色方案]，步骤形状统一，连线箭头清晰，布局整洁有序。
```

## 知识点交互卡片

### 核心要求
- **卡片**：每个知识点一张卡片
- **正面**：知识点标题（8字以内）
- **背面**：知识点详情（20-30字）
- **设计**：统一的卡片风格，有翻转动画效果
- **排列**：网格布局，整齐排列
- **颜色**：每张卡片可以有不同但协调的颜色

### 示例Prompt模板
```
创建一组关于[主题]的知识点交互卡片，包含以下知识点：
1. [知识点1]：[简短说明]
2. [知识点2]：[简短说明]
3. [知识点3]：[简短说明]
4. [知识点4]：[简短说明]
5. [知识点5]：[简短说明]

风格要求：[选定风格]，使用[颜色方案]，卡片大小统一，正面简洁，背面详细，排列整齐。
```

## 知识点时间轴

### 核心要求
- **时间点**：按时间顺序排列的知识点
- **轴线**：清晰的时间轴线
- **标记**：每个时间点有明确标记
- **内容**：每个时间点的知识点说明（15字以内）
- **装饰**：适当的装饰元素，增强视觉效果

### 示例Prompt模板
```
创建一个关于[主题]的知识点时间轴，包含以下时间点：
1. [时间点1]：[知识点1]
2. [时间点2]：[知识点2]
3. [时间点3]：[知识点3]
4. [时间点4]：[知识点4]
5. [时间点5]：[知识点5]

风格要求：[选定风格]，使用[颜色方案]，时间轴线清晰，标记明显，布局整洁有序。
```

## 通用设计原则

1. **简洁明了**：避免过多装饰，突出内容
2. **层次分明**：使用大小、颜色、间距等元素创建视觉层次
3. **一致性**：保持风格、颜色、字体的一致性
4. **可读性**：确保文字清晰可辨，背景与文字对比度适中
5. **信息密度**：合理安排信息密度，避免过于拥挤
6. **视觉平衡**：整体布局平衡，避免偏重一侧
7. **品牌一致性**：如果有品牌标识，保持品牌风格一致

## 输出格式要求

- **文件格式**：PNG或JPG
- **分辨率**：至少1920x1080像素
- **文件命名**：根据类型命名，如 `mindmap.png`、`network.png` 等
- **压缩**：适当压缩，文件大小不超过5MB
- **透明度**：如需透明背景，使用PNG格式
