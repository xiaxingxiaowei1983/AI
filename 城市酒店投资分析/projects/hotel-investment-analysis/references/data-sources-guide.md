## 官方数据源

### 1. 统计公报（最高优先级）

#### 使用场景
- GDP总量及增速
- 三产占比
- 常住人口及就业数据
- 旅游收入数据

#### 检索方式
1. 打开XX市统计局官网
2. 搜索关键词："{城市名} 2024年统计公报"
3. 重点关注：国民经济和社会发展统计公报
4. 数据时间范围：2023-2024年，优先2024年

#### 常用统计局网站
- 北京市统计局：http://tjj.beijing.gov.cn/
- 上海市统计局：https://tjj.sh.gov.cn/
- 广州市统计局：http://tjj.gz.gov.cn/
- 深圳市统计局：http://tjj.sz.gov.cn/
- 杭州市统计局：http://tjj.hangzhou.gov.cn/
- 成都市统计局：http://tjj.chengdu.gov.cn/

### 2. 文旅局数据

#### 使用场景
- 游客接待量
- 热门景点客流
- 旅游收入明细
- 季节分布数据

#### 检索方式
1. 打开XX市文化和旅游局官网
2. 搜索关键词："{城市名} 2024年旅游数据" / "游客接待量"
3. 重点关注：年度旅游统计报告、文旅数据发布
4. 流量数据必须为文旅口公开数据

#### 常用文旅局网站
- 北京市文化和旅游局：http://whlyj.beijing.gov.cn/
- 上海市文化和旅游局：https://wlj.sh.gov.cn/
- 广州市文化广电旅游局：http://wglj.gz.gov.cn/
- 深圳市文化广电旅游体育局：http://wtl.sz.gov.cn/

### 3. 政府官网

#### 使用场景
- 产业政策
- 城市发展规划
- 重大项目信息
- 政策导向

#### 检索方式
1. 打开XX市人民政府官网
2. 搜索关键词："{城市名} 产业发展规划" / "十四五规划"
3. 行业政策必须为政府官网公开链接
4. 重点关注：政策文件、规划文件

## 商业数据源

### 1. 携程/美团

#### 使用场景
- 酒店榜单
- 平均房价（ADR）
- 入住率数据
- 酒店竞争格局

#### 检索方式
1. 打开携程/美团网站或APP
2. 搜索："{城市名} 酒店"
3. 查看热门榜单、价格区间、评价数量
4. 重点关注：2024年数据，按商圈筛选

#### 数据提取要点
- 各档次酒店数量及分布
- 平均房价（按商圈、按档次）
- 入住率（如公开）
- 品牌分布情况

### 2. 高德地图

#### 使用场景
- 客流热力图
- 商圈分布
- 交通枢纽客流
- 区域活跃度

#### 检索方式
1. 打开高德地图
2. 搜索："{城市名} 热力图" / "商圈"
3. 查看客流分布、时间分布
4. 数据来源标注：【来源】高德地图2024年客流数据

#### 数据提取要点
- 核心商圈位置
- 客流热点区域
- 时间分布特征（工作日/周末）
- 交通枢纽位置

### 3. 地铁集团

#### 使用场景
- 地铁客流量
- 站点客流数据
- 交通便捷性分析

#### 检索方式
1. 打开XX市地铁集团官网
2. 搜索："2024年客流数据" / "运营报告"
3. 重点关注：日均客流、站点客流排名

### 4. 券商研报

#### 使用场景
- 酒店行业趋势
- 市场分析报告
- 投资策略建议

#### 检索方式
1. 打开券商研究报告平台（如东方财富、同花顺）
2. 搜索："酒店行业 2024" / "城市酒店投资"
3. 重点关注：主流券商地产/酒店研报

#### 主要券商
- 中信证券、国泰君安、海通证券、华泰证券等

## 租金数据源

### 优先级1：Ai投资指南.Pdf

#### 使用场景
- 核心商圈租金（城市核心商圈租金）
- 机会商圈租金（次核心商圈与普通商圈租金）

#### 匹配规则
- 核心商圈 → 匹配城市核心商圈租金数据
- 机会商圈 → 匹配次核心商圈与普通商圈租金数据

#### 数据优势
- 权威性强
- 数据准确
- 适合投资分析

### 优先级2：中介平台

#### 可用平台
- 链家
- 贝壳找房
- 58同城
- 安居客

#### 使用场景
- 非甲级写字楼租金
- 公寓租金
- 商业综合体租金

#### 检索方式
1. 打开中介平台
2. 搜索："{城市名} 写字楼租金" / "{商圈名} 租金"
3. 按商圈筛选，查看租金范围

#### 注意事项
- 辨别虚高或低价引流情况
- 取中位数或平均数
- 注明数据来源为估算
- 用"≈"表示区间

#### 数据标注格式
```
核心商圈租金 ≈ X.X元/㎡/月（估算逻辑：取链家、贝壳平台中位数）
```

## 数据检索策略

### 检索优先级
1. **官方数据源**（最高优先级）
   - 统计公报
   - 文旅局数据
   - 政府官网政策

2. **商业数据源**（次优先级）
   - 携程/美团榜单
   - 高德客流数据
   - 券商研报

3. **租金数据源**（特殊优先级）
   - 优先：Ai投资指南.Pdf
   - 备选：中介平台

### 数据时间要求
- **优先使用**：2024年全年数据
- **备选使用**：2023年数据
- **数据标注**：必须注明数据年份

### 检索技巧
1. **关键词组合**：
   - "{城市名} 2024年统计公报"
   - "{城市名} 旅游数据 2024"
   - "{城市名} 酒店 价格 2024"
   - "{商圈名} 租金 2024"

2. **数据验证**：
   - 多源交叉验证
   - 官方数据优先
   - 商业数据需辨别真伪

3. **数据保存**：
   - 记录数据来源链接
   - 保存关键数据截图
   - 标注数据采集时间

## 数据标注规范

### 标注格式
每页正文末尾必须标注数据来源：
```
【来源】XX市统计局2024年统计公报
【来源】XX市文化和旅游局2024年旅游数据
【来源】携程酒店榜单2024年数据
【来源】高德地图2024年客流数据
【来源】Ai投资指南.Pdf
【来源】链家、贝壳找房2024年数据
```

### 标注要求
1. **流量数据**：必须为文旅口公开数据
2. **行业政策**：必须为政府官网公开链接
3. **商业数据**：标注平台和数据时间
4. **估算数据**：用"≈"表示，并注明估算逻辑

### 估算数据标注示例
```
核心商圈租金 ≈ 4.5-5.5元/㎡/天
（估算逻辑：取链家平台核心商圈写字楼租金中位数，2024年10月数据）
```

```
次核心商圈租金 ≈ 3.0-4.0元/㎡/天
（估算逻辑：取贝壳找房平台次核心商圈写字楼租金中位数，2024年10月数据）
```

### 禁止行为
1. ❌ 严禁编造、捏造数据
2. ❌ 严禁使用无来源数据
3. ❌ 严禁模糊标注来源
4. ❌ 严禁使用过期数据（2022年及以前）

## 常见问题

### Q1：找不到2024年统计公报怎么办？
A：优先使用2024年上半年数据，如仍无，使用2023年全年数据并标注。

### Q2：文旅局数据不完整怎么办？
A：从统计公报补充旅游收入数据，或从券商研报查找相关数据。

### Q3：中介平台租金数据差异大怎么办？
A：取多个平台的平均值或中位数，用区间表示并注明估算逻辑。

### Q4：高德客流数据如何获取？
A：查看高德地图官方发布的城市客流报告，或使用高德地图热力图功能。

### Q5：如何判断数据是否可靠？
A：
1. 优先官方数据源
2. 多源交叉验证
3. 数据逻辑合理性
4. 与历史数据对比
