# PPT结构模板（参考《属地分析与建议（广州）.ppt》）

## 模板说明
本模板基于《属地分析与建议（广州）.ppt》设计，用于生成简化版PPT。PPT是投资分析报告的精简版，突出核心观点和关键数据，便于汇报和展示。

## PPT设计原则

### 1. 文字简化
- PPT文字约为报告正文的30-50%
- 每页控制在100-150字（不含标题）
- 用要点式表达，避免大段文字

### 2. 突出重点
- 核心数据用图表展示（表格、饼图、柱状图、折线图）
- 关键结论用醒目方式标注（加粗、颜色、图标）
- 数据对比用对比图或差异标注

### 3. 统一风格
- 每页保持一致的视觉层次
- 标题字体：大号、加粗
- 核心数据：中号、醒目
- 辅助说明：小号、简洁

### 4. 观点鲜明
- 每页1个大标题（有观点、有态度）
- 每页1个核心结论（简短有力）
- 结论基于数据，有说服力

## PPT页面结构（P1-P9共16页）

### 通用页面格式
```
大标题（观点性标题，15-20字）

[图表/数据可视化区域]
- 核心数据1
- 核心数据2
- 核心数据3

洞察总结（1句话，20-30字）
```

## PPT制作建议

### 1. 图表选择
- **对比数据**：柱状图、对比图
- **趋势数据**：折线图、面积图
- **占比数据**：饼图、环形图
- **表格数据**：简单表格、高亮关键数据
- **流程展示**：流程图、时间轴

### 2. 颜色使用
- 主色调：商务蓝、稳重灰
- 强调色：橙色、绿色（用于突出关键数据）
- 背景色：白色或浅灰色（保持简洁）

### 3. 字体大小
- 大标题：32-36pt，加粗
- 核心数据：24-28pt，加粗
- 辅助说明：18-20pt，常规
- 图表标签：14-16pt

### 4. 动画效果
- 简单出现动画
- 避免过多动画，保持专业感
- 重点数据可使用强调动画

### 5. 页面布局
- 上部：大标题
- 中部：核心内容（图表+数据）
- 下部：洞察总结

## 与报告的关系

- PPT是报告的精简版，不是替代品
- 报告用于详细分析和归档
- PPT用于汇报和展示
- PPT需与报告数据保持一致
